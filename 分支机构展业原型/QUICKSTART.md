# 快速使用指南

## 🚀 5分钟快速上手

### 第一步：打开系统
双击 `index.html` 文件，在浏览器中自动打开

### 第二步：浏览企业库
1. 点击顶部导航 → **"企业信息"**
2. 查看北京分部维护的4家企业
3. 点击"编辑"查看企业详细信息

### 第三步：创建展业记录
1. 点击顶部导航 → **"新增展业记录"**
2. 填写基本信息：
   - 展业日期：选择日期
   - 分支机构：北京分部（已默认）
   - 负责人员：输入人名
3. 选择企业：点击下拉列表选择已有企业
4. 填写项目说明：描述展业内容
5. 上传留痕：拖拽或点击上传照片/文件
6. 点击"提交保存"

### 第四步：查询展业记录
1. 点击顶部导航 → **"展业查询"**
2. 自动显示所有展业记录列表
3. 可选：添加查询条件筛选
4. 点击"查看"查看详情 + 变更历史

### 第五步：查看统计数据
1. 点击顶部导航 → **"汇总统计"**
2. 查看整体统计数据：
   - 当日新增数量
   - 完成项目数
   - 推进中项目数
3. 查看分支和人员汇总表

## 🔐 权限说明

### 当前用户身份
- 分支：**北京分部**
- 权限：维护北京分部的企业和展业记录

### 能做什么
✅ 查看北京分部的企业列表  
✅ 新增北京分部的企业  
✅ 编辑北京分部的企业  
✅ 删除北京分部的企业  
✅ 新增展业记录  
✅ 查看展业记录  
✅ 编辑展业记录  

### 不能做什么
❌ 查看其他分支的企业  
❌ 编辑其他分支的企业  
❌ 删除其他分支的企业  

### 切换分支用户
如需演示其他分支的权限，修改代码第7行：
```javascript
// 找到这一行
let currentUserBranch = '北京分部';

// 改为其他分支之一
let currentUserBranch = '上海分部';   // 或
let currentUserBranch = '深圳分部';   // 或
let currentUserBranch = '杭州分部';   // 或
let currentUserBranch = '总部';
```

然后刷新浏览器即可看到对应分支的数据

## 💾 数据持久化

### 如何保存数据
- 所有数据自动保存到浏览器的 **localStorage**
- 关闭浏览器后重新打开，数据仍然存在
- 无需连接任何服务器

### 如何清空数据
如需重置为初始示例数据：

**方法1：使用浏览器控制台**
1. 按 F12 打开开发者工具
2. 点击"Console"标签
3. 输入命令：
   ```javascript
   localStorage.clear();
   location.reload();
   ```
4. 按 Enter 执行

**方法2：清除浏览器存储**
1. 打开浏览器设置
2. 清除浏览数据
3. 选中"Cookie和网站数据"
4. 刷新页面

## 🆘 常见问题

### Q1: 数据丢失了怎么办？
A: 数据保存在浏览器的localStorage中。如果：
- 手动清除了浏览器缓存
- 使用了隐私/无痕模式浏览
- 换了浏览器

可以通过刷新页面重新加载初始示例数据。

### Q2: 能否支持多个用户？
A: 当前是演示版本，所有操作共用同一份数据。实际环境需要后端数据库和用户认证。

### Q3: 如何导出数据？
A: 当前版本不支持导出。可以通过浏览器控制台获取数据：
```javascript
// 导出企业信息
console.log(JSON.stringify(companies));

// 导出展业记录
console.log(JSON.stringify(records));
```

### Q4: 支持哪些浏览器？
A: 支持所有现代浏览器（Chrome, Firefox, Edge, Safari等）。不支持IE。

### Q5: 手机可以用吗？
A: 可以，页面是响应式的。但移动端体验会不如电脑端。

### Q6: 企业为什么只能看到一个分支的数据？
A: 这是"谁维护谁查看"的权限设计，确保数据安全。每个分支只能看到自己维护的企业。
