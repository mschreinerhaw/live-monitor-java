# 添加及管理审批规则

资产上下架审批可在资产上架前判断资产的合法性以及资产属性是否完整，避免敏感数据泄露及误用；在资产下架前，评估资产下架后是否影响业务正常运行，为资产管理建立一个结构化和可控的环境，促进资产的有效利用和管理。审批规则能够让您配置哪些资产需要进行上下架审批并可指定审批模板。本文为您介绍如何添加及管理审批规则。

## **权限说明**

超级管理员、运营管理员和具有**上架管理**\-**管理**权限的自定义全局角色支持添加及管理审批规则。

## **上下架审批规则策略**

规则的优先级根据规则排序从上到下递减，单个资产命中的第一个规则即为上架/下架的权限审批规则；若未命中任何规则，则不会发起审批。

## **添加审批规则**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  在左侧导航栏选择**其他设置** > **审批规则**，在**审批规则**页面，单击**添加审批规则**按钮。
    
3.  在**新建审批规则**对话框中，配置参数。业务指标、技术指标、API、仪表板资产与表资产配置一致，下面以**表资产**为例。
    
    | **参数**                  | **描述**                                                     |
    | ------------------------- | ------------------------------------------------------------ |
    | **审批场景**              | 默认为资产上下架，不支持修改。                               |
    | **资产类型**              | 默认为表，不支持修改。                                       |
    | **资产范围**              | 支持为所选资产类型下的**全部**对象或**指定对象**配置审批规则。 - **全部**：针对所选资产类型下全部资产生效审批规则，包括后续新增的资产也会自动执行命中的审批规则。 - **指定对象**：针对审批规则执行时所有资产最新保存的属性值进行匹配，包括后续新增的符合该圈选规则的资产也会自动执行命中的审批规则，规则圈选配置请参见[资产对象范围](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/create-auto-shelf-rules#2b41615e89vk7)。 |
    | **上架模板**/**下架模板** | 选择上架/下架的审批模板，您可以单击**查看模板**，查看审批模板的信息，如需新建审批模板，请参见[创建和管理审批模板](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/template-for-examination-and-approval-of-construction-and-management)。 |
    
4.  单击**确定**，完成审批规则的创建。
    

## **管理审批规则**

业务指标、技术指标、API、仪表板资产与表资产页面展示一致，下面以**表资产**为例。

1.  在审批规则页面，您可以查看已配置审批规则的资产范围、上架/下架的审批模板以及审批规则的最近更新人和最近更新时间信息。
    
2.  您可以对审批规则执行如下操作。
    
    | **操作项** | **描述**                                                     |
    | ---------- | ------------------------------------------------------------ |
    | **排序**   | 单击**排序**按钮，拖拽需要排序的资产范围进行移动，单击**排序完成**按钮，完成排序。此处排序顺序将影响上下架审批规则策略。 |
    | **编辑**   | 支持修改已配置的审批规则信息。                               |
    | **删除**   | 审批规则删除后不可撤销，请谨慎操作。                         |
