# 智能生成

智能生成基于资产元数据信息（资产来源、资产状态），结合大模型能力批量生成资产的系统属性信息（如资产展示名、描述、资产标签、归属目录等），信息确认后一键批量上架，显著提升资产上架配置效率。本文为您介绍如何创建及管理智能生成表、技术指标资产。

## **使用限制**

-   仅表、字段及技术指标资产类型支持智能生成部分系统属性。
    
-   仅保留最近90天的智能生成的任务记录。
    

## **权限说明**

超级管理员、系统管理员及具有**上架管理**\-**管理**权限的自定义全局角色支持新建及管理智能生成任务。

## **创建智能生成任务**

您需要先开启[X-目录管理](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/intelligent-application/)智能应用，才能创建智能生成任务。

1.  在Dataphin首页的顶部菜单栏选择**超级X** > **智能应用管理**。
    
2.  在左侧导航栏单击**智能应用**，在**智能应用**页面，完成[模型调用配置](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/intelligent-application/)后，开启X-目录管理智能应用。
    
3.  在Dataphin首页的顶部菜单栏，单击**资产** > **目录管理**。
    
4.  在左侧导航栏选择**自动上架** > **智能生成**，在**智能生成**页面，单击右上角的**新建智能生成任务**按钮。
    
5.  在**新建智能生成任务**对话框中，配置参数。
    
    | **参数**         |                                                              | **描述**                                                     |
    | ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
    | **基本信息**     | **任务名称**                                                 | 输入智能生成的任务名称，全局唯一，不超过128个字符。          |
    | **资产圈选**     | **资产类型**                                                 | 选择需要智能生成的资产类型，支持**表**、**指标**资产。       |
    | **资产范围**     | 针对选中的指定对象，支持根据资产状态（暂不上架不支持该条件）、资产全名、资产名称、资产展示名、资产来源、资产标签圈选资产范围，以便对资产对象范围进行更细粒度过滤，筛选出需要进行智能生成的资产。 - **资产状态**（暂不上架不支持该属性）、**资产来源**：支持的筛选条件为属于、不属于。 **属于**/**不属于**：当资产属性为资产状态时，需选择资产的汇总状态，支持准资产（包括未上架、上架失败、暂不上架失败）、已上架资产（包括已上架、更新待上架、更新上架失败、下架失败）；当资产属性为资产来源时，可选择智能生成的资产范围对应的资产来源。 - **资产全名**、**资产名称**、**资产展示名**：支持的筛选条件为前缀匹配、后缀匹配、包含、属于、为空（仅资产展示名支持该条件）。 - **前缀匹配**、**后缀匹配**、**包含**：支持输入不超过256个字符。 - **属于**：支持选择不超过500个资产。 - **为空**：无需配置。 - **资产标签**：支持的筛选条件为包含任一、包含所有，具有**上架管理**\\-**管理**权限的自定义全局角色，支持单击**标签管理**按钮，跳转至**资产** > **目录管理** > **其他设置** > **基础设置**页面新建标签。 - **包含任一**：资产标签需包含所选的任意一个标签值即为满足条件。 - **包含所有**：资产标签需包含所选的所有标签值即为满足条件。 **说明** - 最多配置10个过滤条件。 - 过滤条件之间的关系可配置为且、或。 |                                                              |
    | **属性配置**     | **表属性**/**字段属性**/**指标属性**                         | 选择需智能生成的资产系统属性。当资产类型为表时，支持选择**表属性**、**字段属性**、**指标属性**（仅支持指标上架管理时展示）；当资产类型为指标时，支持选择**指标属性**。 |
    | **自动识别指标** | 当资产类型为表时可配置。默认关闭，关闭后，将不会自动识别表中可能存在的自定义指标，若同时选择指标属性，仅针对表中已存在的建模指标和自定义指标智能生成配置选中的系统属性信息。 开启后，将自动识别表中可能存在的自定义指标，若同时选择指标属性，将针对所有指标智能生成配置选中的系统属性信息。 **说明** - 系统仅针对支持自定义指标的表（除了逻辑表及逻辑视图）进行自动识别。 - 单表生成属性时需手动选择数据板块；批量生成时，若表存在未配置的数据板块，系统将跳过不识别，可手动配置后再进行识别。 |                                                              |
    
6.  单击**确定**，完成新建智能生成任务的创建。
    

## **管理智能生成任务**

1.  智能生成页面为您展示已创建的任务，包括任务名称/资产类型、生成属性、资产总数、生成进度、审核状态以及任务的创建人和创建时间。
    
    **资产总数**：表资产仅统计表个数，不统计指标个数。
    
2.  您可以根据任务的名称进行搜索，也可根据资产类型、创建人、生成进度进行筛选，或快捷筛选**我创建的**的任务。
    
3.  在目标任务的操作列下可进行如下操作。
    
    -   **查看**：单击跳转至任务详情页面，查看生成成功和未生成成功的资产并执行相关操作，详情请参见下方的**智能生成任务列表**。
        
    -   **删除**：当任务的生成进度为生成中时不展示该操作。任务删除后，已生成的内容将全部清除，删除后不可撤销，请谨慎操作。

| ②**搜索与搜索** | - **生成成功**：您可以根据资产全名进行搜索，也可根据资产来源、审核状态进行筛选。 - **未生成成功**：您可以根据资产全名进行搜索，也可快捷筛选待生成、生成中、已终止、生成失败的资产。 |
| ③**任务列表**   | 为您展示生成成功以及未生成成功列表。单次批量操作最多可选择10000条资产。 - **生成成功**：展示圈选资产所有需要生成且均已成功产出的结果，包括待审核、或置为已应用、已弃用的资产，其中，已应用/已弃用状态的资产为最新保存的属性值（可能和操作时刻的结果不同）。您可以对资产执行编辑、应用、弃用、重新生成操作。 - **未生成成功**：展示圈选资产智能生成未成功的资产，包括待生成、生产中、生成失败以及手动终止的资产。您可以对资产执行终止、重新生成操作。 ## 生成成功 为您展示资产的名称/来源、展示名、归属目录、标签、描述、字段/指标、审核状态以及最近上架状态信息。其中，展示名、归属目录、标签、描述根据生成任务配置决定展示列，仅展示需要智能生成的属性。 **展示名**、**归属目录**、**标签**、**描述**：当审核状态为待审核时，第一行展示当前资产最新保存的对应属性值，第二行展示智能生成的属性值；当审核状态为已应用/已弃用时，展示当前资产最新保存的对应属性值。 同时，您可对资产执行如下操作。 - **编辑**：仅上架状态支持编辑的资产可操作。单击除操作列外的任意位置或单击操作列下的**编辑**图标，打开当前资产的**智能属性生成**面板，查看智能生成的属性信息，默认仅展示资产名称和需要智能生成的属性信息，如果您需要查看或编辑更多内容，可取消选中右上角**仅展示智能生成属性**，整体信息确认无误后再进行应用或上架操作，详情请参见[智能生成资产详情](#cac3251af4uvf)。 - **应用**：仅待审核状态支持操作。单击操作列下的**应用**或选中资产后，单击底部的**应用**，即使用本次智能生成的信息。应用会以最新智能生成的内容为准对资产进行保存修改操作，仅支持编辑的资产可执行应用操作。 - **弃用**：仅待审核状态支持操作。单击操作列下的**弃用**或选中资产后，单击底部的**弃用**，即放弃本次智能生成的信息不使用。此外，您也可以直接重新生成信息。 - **编辑属性**：选中资产后，单击底部的**编辑**属性，为所选资产对象完善属性信息，详情请参见[修改属性](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#8f59e9e199gx6)。 - **重新生成**： - **单个操作**：单击操作列下的更多，选择**重新生成**，将按照任务配置重新智能生成相关属性，并自动将该资产从生成成功列表移至未生成成功列表。 - **批量操作**：选中资产后，单击底部的**重新生成**，在**针对已选资产进行重新生成**对话框中，按审核状态选择需要重新生成的资产。 - **上架：**同**手动上架** > **准资产**列表的上架操作，详情请参见[上架](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#56a4e7fd43iqe)。 ## 未生成成功 为您展示资产的名称、资产来源、生成状态以及失败原因信息。 同时，您可对资产执行如下操作。 - **终止**：当生成状态为生成中时支持操作。单击操作列下的**终止**或选中资产后，单击底部的**终止**，停止正在生成中或待生成的属性信息，后续可重新生成。 - **重新生成**：当生成状态为已终止或生成失败时支持操作。单击操作列下的**重新生成**或选中资产后，单击底部的**重新生成**，将按照任务配置重新智能生成相关属性。生成中的资产将自动跳过不处理。 |

#### **智能生成资产详情**

配置信息与准资产的编辑时的[智能生成](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#c982e0474ems3)相似，不同详情见下方表格。

| **参数**     | **描述**                                                     |
| ------------ | ------------------------------------------------------------ |
| **基本信息** | 支持修改已智能生成的属性信息。 **标签**：具有**上架管理**\\-**管理**权限的自定义全局角色，支持单击**标签管理**按钮，跳转至**资产** > **目录管理** > **其他设置** > **基础设置**页面新建标签。 |
| **字段信息** | 根据创建智能生成任务时配置的属性展示对应的列。若开启自动识别指标，可为字段配置业务类型。特殊的，若表存在未配置的数据板块或数据板块已删除，系统将跳过不识别，可手动配置后再进行识别。 |
| **相关指标** | 仅自定义指标资产支持展示，根据创建智能生成任务时配置的属性展示对应的列。支持单击目录、标签后的**清空**，一键清除智能生成的信息。 若选择**表属性覆盖未上架指标属性**，则保存表资产时，未上架指标的展示名称、描述、可见范围、数据探查报告（如有）、血缘关系、质量概况（如有）、资产维护人员等属性将使用表属性覆盖更新。 |

确认全部信息无误后，单击**应用**，使用此次智能生成的属性信息。

您可根据资产的不同审核状态对资产详情执行不同的操作。

**待审核**：弃用、应用、重新生成、应用并上架。

**已生成**/**已弃用**：取消、保存、重新生成、保存并上架。

-   **弃用**：不使用此次智能生成的属性信息。
    
-   **重新生成**：重新生成智能生成的属性信息。
    
-   **继续生成**：仅已终止资产支持操作。将继续智能生成停止的属性信息。
    
-   **应用并上架**：使用此次智能生成的属性信息并将资产上架到目录前台展示，上架操作同**手动上架** > **准资产**列表一致，详情请参见[上架](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#56a4e7fd43iqe)。
    
-   **取消**/**保存**：取消或保持此次智能生成的属性信息。
    
-   **保存并上架**：保存此次智能生成的属性信息并将资产上架到目录前台展示，上架操作同**手动上架** > **准资产**列表一致，详情请参见[上架](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#56a4e7fd43iqe)。
