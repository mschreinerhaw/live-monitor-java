# 创建自动上架规则

资产对象根据规则组配置的时间按照规则定时执行暂不上架、自动上架或完善属性信息，从而降低人力管理成本。本文为您介绍如何创建自动上架规则。

## **使用限制**

-   单个租户下支持创建不超过500条规则组。
    
-   每个规则组下最多创建50条规则。
    

## **权限说明**

-   超级管理员、运营管理员和拥有**上架管理**\-**管理**权限的自定义全局角色支持创建规则组、规则。
    
-   规则负责人支持在指定规则组下创建及管理规则。
    

## **新建规则组**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  在左侧导航栏选择**自动上架** > **自动上架规则**，在**自动上架**页面，单击**新建规则组**按钮。
    
3.  在**新建规则组**对话框中，配置参数。
    
    | **参数**       | **描述**                                                     |
    | -------------- | ------------------------------------------------------------ |
    | **规则组名称** | 规则组的名称，全局唯一，不超过256个字符。                    |
    | **规则负责人** | 规则组下的规则负责人，可选择当前租户下正常使用的用户，负责规则的常态化管理。 |
    | **资产类型**   | 选择需要进行规则执行的资产类型，支持选择**表**、**技术指标**、**API**、**仪表板**资产。 |
    | **资产来源**   | 根据资产类型圈选对象范围，规则针对所选的资产对象生效。每个资产来源仅能唯一归属一个资产组。 |
    | **执行方式**   | 规则组下的所有规则将根据此处配置按照顺序自动执行，当前支持**定时执行**，支持**每日**、**每月**，每日的默认时间为6:00，支持修改，可配置的定时开始执行时间区间为00:00～23:59。 |
    
4.  单击**确定**，完成规则组的创建，然后可以在规则组下创建规则。
    

## **新建规则**

1.  在**自动上架**页面，单击右上角的**新建规则**按钮或单击目标规则组**操作**列下的**新建规则**。
    
2.  在**新建自动上架规则**页面，配置参数。
    
    | **参数**             |                                                              | **描述**                                                     |
    | -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
    | **基本信息**         | **所属规则组**                                               | 选择规则归属的规则组，在当前规则组下创建规则时默认填充当前规则组，支持修改。 |
    | **规则名称**         | 规则的任务名称。不超过256个字符，所属规则组下名称唯一。      |                                                              |
    | **规则类型**         | 选择规则的类型，支持**自动上架**、**信息完善**、**暂不上架**。 |                                                              |
    | **资产对象范围**     | **资产类型**                                                 | 根据所选规则组展示资产类型。                                 |
    | **资产范围**         | 根据所选规则组展示资产范围。                                 |                                                              |
    | **资产对象**         | 支持为所选规则组下的**全部**对象或**指定对象**配置规则。 - **全部**：所属规则组下指定的资产范围，包括后续新增的资产。 - **指定对象**：针对规则执行时所有资产最新保存的属性值进行匹配，包括后续新增符合圈选规则的资产。 |                                                              |
    | **过滤条件**         | 针对选中的指定对象，支持根据资产状态（暂不上架不支持该条件）、资产全名、资产名称、资产展示名、资产来源、资产标签、所属BI空间/目录配置过滤条件，以便对资产对象范围进行更细粒度过滤，筛选出需要进行规则执行的资产。 - **资产状态**（暂不上架不支持该属性）、**资产来源、所属BI空间/目录**（仅仪表板资产支持该属性）：支持的筛选条件为属于、不属于。 **属于/不属于**：当资产属性为资产状态时，需选择资产的汇总状态，支持准资产（包括未上架、上架失败、暂不上架失败）、已上架资产（包括已上架（仅信息完善支持）、更新待上架、更新上架失败、下架失败）；当资产属性为资产来源或所属BI空间/目录时，可选择规则组所圈选的资产范围对应的资产来源或所属BI空间/目录。 - **资产全名**（仪表板资产不支持该属性）、**资产名称**、**资产展示名**（API资产、仪表板资产不支持该属性）：支持的筛选条件为前缀匹配、后缀匹配、包含、属于、为空（仅资产展示名支持该条件）。 - **前缀匹配**、**后缀匹配**、**包含**：支持输入不超过256个字符。 - **属于**：支持选择不超过500个资产。 - **为空**：无需配置。 - **资产标签**：支持的筛选条件为包含任一、包含所有。 - **包含任一**：资产标签需包含所选的任意一个标签值即为满足条件。 - **包含所有**：资产标签需包含所选的所有标签值即为满足条件。 配置说明： - 至少配置一条规则。 - 最多配置10个过滤条件。 - 过滤条件之间的关系可配置为且、或。 |                                                              |
    | **信息完善配置**     |                                                              | 规则类型选择信息完善时需配置。为所选资产对象完善属性信息，详情请参见[修改属性](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#8f59e9e199gx6)。 |
    | **暂不上架配置**     |                                                              | 规则类型选择暂不上架时需配置。配置暂不上架原因及说明，将无需上架的资产指定为暂不上架资产，详情请参见[暂不上架](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#c0b94ca33cn74)。 |
    | **执行配置**         | **是否覆盖手动操作**                                         | 支持选择**是**或**否**。 - **信息完善**：选择**是**，针对规则命中的所有资产对象，均按照规则配置更新属性值；选择**否**，若需要修改的属性的最近一次值变更来自手动编辑，则会跳过不更新，保留手动编辑的属性值。 - **自动上架**：选择**是**，针对规则命中的所有资产对象执行上架操作；选择**否**，若资产最近一次状态变更来自**手动下架**，则跳过不更新。 - **暂不上架**：选择**是**，针对规则命中的所有资产对象执行暂不上架操作；选择**否**，若资产最近一次状态变更来自**手动变更为准资产**，无论是否变更成功，均跳过不更新。 |
    | **是否跳过上架审批** | 支持选择**是**或**否**。 选择**是**，无论资产是否命中上架审批规则，都跳过上架审批；选择**否**，命中上架审批规则的资产，需经过审批后才能上架。 |                                                              |
    
3.  单击**仅保存**，需在自动上架页面手动开启生效状态。单击**保存并生效**，规则将按照规则组配置执行。
    
    自动上架规则根据规则组的配置按照顺序自动执行，不校验规则负责人是否拥有资产对象的维护权限。
    

## **后续步骤**

-   规则执行成功后，您可以在资产列表进行查看，详情请参见[管理准资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets)。
    
-   您可对规则组、规则执行编辑、删除等操作，详情请参见[管理自动上架规则](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/manage-automatic-shelving-rules)。
