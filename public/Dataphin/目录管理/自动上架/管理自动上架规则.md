# 管理自动上架规则

您可以对已创建的规则组、规则执行编辑、删除等操作。本文为您介绍如何管理自动上架规则。

## **权限说明**

-   超级管理员、运营管理员和拥有**上架管理**\-**管理**权限的自定义全局角色支持管理规则组、规则及查看规则执行明细。
    
-   规则负责人支持在指定规则组下管理规则及查看规则执行明细。
    

## **操作步骤**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  在左侧导航栏选择**自动上架** > **自动上架规则**，进入**自动上架**页面。
    
    自动上架页面为您展示规则组/规则的名称、类型、资产来源、最近执行状态、规则组及规则的负责人/最近更新时间、生效状态信息。其中，当前规则所在规则组手动执行了部分规则，则展示最近1次手动执行的状态。
    
3.  （可选）您可以根据规则名称或规则组名称搜索目标资产，也可以根据规则类型、资产类型、资产来源、规则生效状态、最近执行状态、执行结束时间筛选目标资产。
    
    您可单击**说明**按钮，查看资产上架说明。
    
4.  在目标资产操作列下可进行如下操作。
    
    | **操作项**               | **描述**                                                     |
    | ------------------------ | ------------------------------------------------------------ |
    | **规则组操作**           |                                                              |
    | **查看**                 | 若无规则组编辑权限，支持查看操作。单击操作列下的**查看**，可查看已配置的规则组信息，如需编辑，可单击底部的**编辑**按钮，快捷修改信息。 |
    | **编辑**                 | 单击操作列下的**编辑**，支持修改除资产类型之外的其他信息，若修改了资产来源，将影响已建规则资产来源为全部的对象范围；若切换规则组，已配置信息将全部清空且无法恢复，请评估后进行操作。 |
    | **新建规则**             | 单击操作列下的**新建规则**，可快捷进入该规则组下的新建规则页面配置规则。 |
    | **排序**                 | 您可以单击操作列下的**排序**，在**规则排序**对话框中，拖动规则进行排序，单击**确定**，完成排序。 **说明** - 排序顺序将影响该规则组下规则的执行顺序。 - 若有一个正在执行中的任务，此次排序不会影响已生成的任务，而是会根据执行时刻记录的快照顺序执行。 |
    | **手动执行**             | 如果需要临时运行规则，您可以单击操作列下的**更多**\\-**手动执行**。 当最近执行状态为**等待中**、**执行中**时不支持操作；当前规则组下没有**生效**状态的规则时不支持操作。 **说明** 如果到达下一次定时运行时间时本次运行仍未结束，则下一次定时执行任务将会被跳过。 |
    | **终止执行**             | 如果需要临时终止规则运行，您可以单击操作列下的**更多**\\-**终止执行**。 当规则组最近一次执行状态为**等待中**、**执行中**，支持终止执行。 **说明** 规则组终止执行后，已操作完成的资产变更将不会回退，请确认资产状态是否符合预期。 |
    | **查看最近10次执行明细** | 您可以单击操作列下的**更多**\\-**查看最近10次执行明细**，在**执行明细**对话框中，每个资产相关对应每一条规则的执行结果为1条记录，您可以筛选具体规则查看该规则覆盖的所有资产执行情况，也可以筛选具体资产查看在规则组的整体规则作用下的变更记录。为您展示资产类型、资产范围、执行结果、资产执行明细信息。同时，您可以执行编辑资产、查看规则、筛选等操作。 - **执行明细列表**：为您展示资产执行结束后的执行记录明细，包括资产的名称、所属表（仅指标资产支持该信息）、资产来源、执行结果、执行后状态、说明、执行规则、执行结束时间。 - **执行记录支持的操作**： - **执行结果**：当执行记录选择具体某一执行结束时间时，支持查看执行结果，包括本次执行时资产的总数以及执行结果记录数。 - **执行记录**：系统会为您筛选最近10次规则组执行结束时间的记录，您也可以在执行记录切换查看具体某一执行结束时间的执行记录。 - **筛选**：支持快捷筛选**失败**、**跳过**的记录，或根据规则、资产、所属表（仅指标资产支持该筛选）、执行结果、执行后状态进行筛选。 - **编辑资产**：单击资产操作列下的**编辑资产**图标，跳转至当前资产的编辑页面修改资产信息，若资产已删除或状态为暂不上架，不支持修改信息。 - **查看规则**：单击资产操作列下的**查看规则**图标，跳转至当前规则的查看页面。 |
    | **删除**                 | 您可以单击操作列下的**更多**\\-**删除**，规则组删除将同步删除该规则组下的所有规则，正在执行中的规则将会立即终止。 |
    | **查看最近1次执行明细**  | 任务执行结束后，您可以单击最近执行状态列下的**查看最近1次执行明细**图标，在**执行明细**对话框中，系统会为您筛选对应规则的最近1次执行结束时间的执行记录。展示的明细信息和支持的操作与查看最近10次执行明细一致。 |
    | **规则操作**             |                                                              |
    | **查看**                 | 单击规则的名称，可查看已配置的规则信息，如需编辑，可单击底部的**编辑**按钮，快捷修改信息。 |
    | **编辑**                 | 单击操作列下的**编辑**，支持修改配置信息。                   |
    | **手动执行**             | - **单个手动执行**：如果需要临时运行规则，您可以单击操作列下**手动执行**。 - **批量手动执行**：针对所选规则组下的规则进行批量操作。选中需要临时运行规则，单击底部的**手动执行**，运行规则。 **说明** - 规则组支持整组执行，如果您选择对单条规则进行操作，本质上是将所属规则组下其他规则跳过不操作，执行状态和执行结果仍按规则组统计。 - 当规则组最近执行状态为**等待中**、**执行中**时，不支持手动执行；当前规则组下没有**生效**状态的规则时不支持操作。 - 批量手动运行规则时，仅对选中的有操作权限且最近执行状态支持手动执行的规则进行操作，所属规则组下其他未选中的规则将会被跳过。 - 如果到达下一次定时运行时间时本次运行仍未结束，则整个规则组的下一次定时执行任务将会被跳过。 |
    | **终止执行**             | - **单个终止执行**：如果需要临时终止规则运行，您可以单击操作列下的**终止执行**。 - **批量终止执行**：针对所选规则组下的规则进行批量操作。选中需要临时终止运行的规则，单击底部的**手动终止**，停止规则的运行。 **说明** - 当规则组最近执行状态为**等待中**、**执行中**时，支持终止执行。 - 规则组支持整组终止，如果您选择单条规则进行操作，若同一规则组下还有其他执行中的规则，将被同步终止。 - 批量手动终止规则时，仅对选中的有操作权限且最近执行状态支持手动终止的规则组进行操作，其他选中的规则组系统将跳过不操作。 - 规则组终止执行后，已操作完成的资产变更将不会回退，请确认资产状态是否符合预期。 |
    | **查看最近10次执行明细** | 您可以单击操作列下的**更多**\\-**查看最近10次执行明细**，在**执行明细**对话框中，为您展示资产最近10次执行结束时间的执行记录。展示的明细信息和支持的操作与规则组的查看最近10次执行明细一致。 |
    | **克隆**                 | 单击操作列下的**克隆**图标，支持快捷复制规则的配置信息。     |
    | **删除规则**             | - **单个删除**：单击**操作**列下的**删除**，可删除规则。 - **批量删除**：针对所选规则组下的规则进行批量操作。选中需要删除的规则，单击底部的**删除规则**，可删除规则。 **说明** 规则删除不会影响正在执行中或等待中的任务，状态生效的规则仍会继续执行。 |
    | **置为生效**             | - **单个规则置为生效**：单击**生效状态**列下的开关，开启生效状态。 - **批量置为生效**：针对所选规则组下的规则进行批量操作。选中需要开启生效状态的规则，单击底部的**生效**，开启生效状态。 |
    | **置为不生效**           | - **单个规则置为不生效**：单击**生效状态**列下的开关或底部的**置为不生效**，关闭生效状态。 - **批量置为不生效**：针对所选规则组下的规则进行批量操作。选中需要关闭生效状态的规则，单击底部的**不生效**，关闭生效状态。 **说明** 关闭后，正在执行中或等待中的任务仍会继续执行，后续将会跳过该规则不执行。 |
