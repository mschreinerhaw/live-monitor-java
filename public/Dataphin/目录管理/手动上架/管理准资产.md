# 管理准资产

准资产列表为您展示通过采集适配器指定数据源采集的资产和从数据中台获取的资产。本文为您介绍如何查看及管理准资产。

## **权限说明**

-   超级管理员、运营管理员和具有**上架管理-管理**权限的自定义全局角色支持新建、上架及管理准资产。
    
-   业务系统的运营人员、具有资产维护权限的维护人员支持管理自己所负责的业务系统下的资产。
    

## **操作说明**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  单击左侧导航栏**手动上架** > 对应资产类型，进入**准资产**页面。
    
    准资产列表为您展示通过采集适配器指定数据源采集的资产和从数据中台获取的资产，包括：资产名称、指标类型、关联技术指标、资产来源、归属目录、最新状态、信息是否完善、是否曾上架、资产引入时间、最新数据时间、最近更新时间等信息。
    
    -   **资产来源**：为您展示资产对象所属的来源；其中：Dataphin数据板块下仅包含逻辑表；API资产的资产来源为数据服务项目；Quick BI仪表板的资产来源为Quick BI工作空间。
        
    -   **最新状态**：针对最新状态为上架失败的资产，您可以将鼠标悬停状态后的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0746511471/p913068.png)图标，查看上架失败的原因，修改后重新上架资产。
        
    -   **信息是否完善**：为您展示资产的必填属性是否填写完整及是否合法。
        
    -   **是否曾上架**：为您展示资产是否下架后再上架。
    
3.  （可选）您可以根据指标类型、资产来源、BI空间/目录、最新状态、归属目录、信息是否完善、是否曾上架、资产引入时间、最新数据时间、最近更新时间进行精确筛选，也可以根据指定条件搜索资产。
    
4.  在目标资产操作列下可进行如下操作。不同类型资产可执行的操作相同，下面以**表资产**为例进行说明。单次批量操作最多可选择10000条资产。
    
    | **操作项**       | **描述**                                                     |
    | ---------------- | ------------------------------------------------------------ |
    | **编辑**         | 支持修改资产的基本信息、使用说明以及配置资产的可见权限等信息，详情请参见[编辑资产信息](#6ca7f69f10265)。 |
    | **上架**         | 单击**上架**或选中资产后，单击底部的**上架**，可将资产上架到资产目录前台展示。若当前资产配置了审批规则，则上架需要经过审批，审批备注不超过100个字符，审批通过后可上架到资产目录前台；若当前资产未配置审批规则，则可直接上架。 **说明** - 仅必填信息配置完整的资产支持上架。 - 当表资产上架后，该表下的自定义指标资产才能上架。 - 仅具有资产维护权限的用户及角色可上架资产。 - 仅最新状态为未上架、上架失败的准资产支持上架。 - 已上架状态的业务指标名称需在租户内唯一。 |
    | **暂不上架**     | 单击**暂不上架**并填写暂不上架原因，可将资产状态修改为暂不上架。暂不上架的资产不会在资产目录前台展示。 如您需要编辑暂不上架资产，请先将其修改为准资产状态。 **说明** 暂不上架表资产操作将同步对其包含的自定义指标生效，但会跳过无维护权限的指标。上架自定义指标时，如所属表状态为暂不上架，需先将表资产变更为准资产后，再上架指标资产。 |
    | **设置维护权限** | 业务指标不支持该操作。 - **单个操作**：单击**设置维护权限**，支持配置资产的维护人员，负责资产的日常维护，包括修改信息及上下架。 - **批量操作**：单击底部**设置维护权限**，支持批量追加或修改维护人员。 - **修改**：将选中资产的维护人员替换为本次指定的维护人员。 - **追加**：保留选中资产的已配置的维护人员并追加本次指定的维护人员。 |
    | **查看变更记录** | 为您展示单个资产最近30天的变更记录，包括属性变更、状态变更等，以便追溯。 **说明** 对于自定义指标，统计粒度变更对应所属表中定义为维度的字段变更，且跟随表上架操作生效。 |
    | **删除**         | 仅业务指标支持该操作。当业务指标的状态为未上架、暂不上架失败、上架失败时，支持删除业务指标，并同步取消相关业务指标和相关技术指标的关联关系。 |
    | **修改属性**     | 选中资产后，单击底部的**修改属性**，在批量修改资产属性页面，修改资产的属性。 与编辑单个资产相比，部分属性支持更灵活的配置。 - **添加所有必填属性**：单击可快速添加当前资产类型上架必须配置的属性。 - **表展示名称**/**描述**：支持引用DDL注释和自定义，可选是否**仅对展示名称为空的资产生效**，若未选中，将覆盖更新选中表已配置的展示名。 - **引用DDL注释**：将引用数据表的注释（comment）作为资产的展示名称/描述，并跟随DDL（comment）变更而自动更新。 - **自定义**：支持选择**表**/**字段名称**或**DDL注释**作为资产的展示名称；支持选择**DDL注释**作为资产的描述或将描述**置空**。 - **字段展示名称**/**描述**：支持引用DDL注释和自定义，可选是否**仅对展示名称为空的资产生效**，若未选中，将覆盖更新选中表已配置的展示名。 - **引用DDL注释**：若选中**仅对展示名称为空的资产生效**，则针对当前表中所有字段为空的展示名称/描述生效。将引用数据字段的注释（comment）作为资产的展示名称/描述，并跟随DDL（comment）变更而自动更新。 - **自定义**：若选中**仅对展示名称为空的资产生效**，则针对当前表中单个字段为空的展示名称/描述生效。支持选择**字段名称**或**DDL注释**作为资产的展示名称；支持选择**DDL注释**作为资产的描述或将描述**置空**。 **说明** - 字段的展示名称/描述不论是引用DDL注释还是自定义，都需要将当前表的全部字段统一配置为引用DDL注释或自定义。 - 若将字段的展示名/描述从引用DDL注释改为自定义且选中仅对展示名称为空的资产生效，则当前表所有字段的DDL注释都为空才会生效。 - 若将字段的展示名/描述从自定义改为自定义且选中仅对展示名称为空的资产生效，则当前表的单个字段为空即生效。 - **归属目录/资产标签/相关业务指标**：支持在已有配置上**追加**更新，或按照最新当前配置**修改**已有配置。 当前账号具有**上架管理**\\-**管理**权限的，支持单击**标签管理**按钮，跳转至**资产** > **目录管理** > **其他设置** > **基础设置**页面新建标签。 - **业务负责人**：仅业务指标支持修改业务负责人。 - **可见范围**/**数据探查报告**：控制是否在资产目录的资产详情页展示对应内容。 - **原子指标**/**业务限定**/**统计周期**/**度量单位**：仅自定义技术指标支持修改。 - **是否可累加**/**关联业务指标**：仅技术指标支持修改。 - **血缘关系**/**质量概况**：仅表资产和指标资产支持修改，控制是否在资产目录的资产详情页展示对应内容。 - **指标关系图**：仅规范建模指标资产支持修改，控制是否在资产目录的资产详情页展示对应内容。 - **元数据变更记录**：仅表资产支持修改，控制是否在资产目录的资产详情页展示对应内容。 - **其他自定义属性**：您可在**规划** > **属性管理**中为资产配置其他自定义属性，以丰富资产信息。 **说明** - API资产、仪表板资产的系统属性支持修改归属目录、资产标签、可见范围。 - 业务指标资产支持修改归属目录、资产标签、业务负责人、可见范围、相关业务指标。 - 当表类型为物理表、物理视图、物化视图、数据源表时，表和字段的展示名称、描述支持引用DDL注释作为资产属性。 |
    | **智能生成**     | 仅表/技术指标资产支持该操作。针对可编辑的资产进行智能生成，选中需智能生成的资产，单击底部的**智能生成**，在新建智能生成任务对话框中，配置信息，详情请参见[智能生成](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/intelligent-generation)。 |
    | **导出资产**     | 单击底部的**导出资产**，支持批量导出列表选中的资产。对于当前操作人没有维护权限的资产，系统会自动跳过。 如果导出的资产内容较多，文件生成需要等待一段时间，您可以在**批量管理**\\-**批量操作记录**中查看本次导出操作信息，并下载导出文件。 |
    
    **说明**
    
    -   部分资产属性的变更需重新上架才会更新目录前台的展示，如：归属目录、标签；部分属性变更则无需重新上架，即目录前台仅展示最新的属性（如字段、仪表板图表信息等）
        
    -   若已上架资产被删除，则**需重新上架**列会展示为**是**，建议您重新上架以保持数据一致性和可用性。
        
    -   若资产被删除，仅支持查看。
        
    

### **编辑资产信息**

**说明**

-   表资产保存后，相关指标将自动生成对应的指标资产。
    
-   在底部选中**保存并上架时，同步上架相关指标**，则会同步上架表资产和相关指标资产。
    
-   请先上架表，再上架表的自定义指标。
    
-   当表类型为物理表、物理视图、物化视图、数据源表时，表和字段的展示名称、描述支持引用DDL注释作为资产属性。
    

### **表资产**

| **参数**             |                                                              | **描述**                                                     |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **基本信息**         | **名称**                                                     | 为您展示资产的名称，不支持修改。                             |
| **展示名称**         | 填写资产的展示名称，支持引用DDL注释和自定义展示名称。 - **引用DDL注释**：将引用数据表的注释（comment）作为资产的展示名称，并跟随DDL（comment）变更自动更新。 - **自定义**：输入展示名称，不超过256个字符。您也可以单击**填入资产名称**或**填入资产描述**，快捷填充资产的名称或描述作为展示名。 |                                                              |
| **归属目录**         | 选择表专题下已发布的目录，支持多选。                         |                                                              |
| **标签**             | 支持选择不超过20个标签，当前账号具有**上架管理**\\-**管理**权限，支持单击**标签管理**按钮，跳转至**资产** > **目录管理** > **其他设置** > **基础设置**页面新建标签。 |                                                              |
| **描述**             | 支持引用DDL注释和自定义描述。 - **引用DDL注释**：将引用数据表的注释（comment）作为资产的描述，并跟随DDL（comment）变更而自动更新。 - **自定义**：填写资产的相关描述，不超过1000个字符。 |                                                              |
| **属性信息**         | **业务属性**/**管理属性**/**技术属性**                       | 为您展示资产的系统属性及自定义属性信息，您可以在右上角选中**隐藏只读属性**，聚焦于需编辑的信息。 - **系统属性**： - **业务属性**：包括所属主题域信息。 - **管理属性**：包括归属系统信息。 - **技术属性**：包括资产来源、所属板块/项目空间、数据库类型、所属的项目、表类型、资产引入名称、采集适配器类型、采集适配器信息。 - **自定义属性**：填写规范取决于表属性的配置。例如：变更管理所对应的填写方式为下拉单选，可选范围为是、否。因此，编辑表资产时，该属性值为下拉单选框，可选枚举值范围为是、否。更多属性配置，请参见[创建及管理资产对象、API、仪表板属性](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/create-and-manage-r-d-object-properties)。 当属性取值为超链接时，链接前缀需为`http://`和`https://`才能使链接生效，不支持仅输入空格，字符长度取决于属性的配置，最大不超过10000个字符。 |
| **字段信息**         | **序号**                                                     | 您可以单击序号后的**排序**![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3372971271/p807599.png)图标，字段将根据序号进行排序。 |
| **字段名**           | 您可以单击字段名后的**排序**![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3372971271/p807599.png)图标，字段将根据字段名称进行排序。 |                                                              |
| **数据类型**         | 为您展示字段的数据类型。                                     |                                                              |
| **字段展示名**       | 填写字段的展示名称，支持引用DDL注释和自定义描述。 - **引用DDL注释**：将引用字段的注释（comment）作为资产的展示名称，并跟随DDL（comment）变更而自动更新。 - **自定义**：输入字段的展示名称，不超过512个字符。您可将鼠标悬停**快速填充**，选择**填入描述**或**填入名称**，快捷填充字段的名称或描述作为展示名称。 |                                                              |

| **业务类型**         | 支持选择维度、指标、统计周期三种业务类型。                   |                                                              |
| **关联实体**         | - 当表类型为物理表、物理视图、视图且字段的业务类型为维度时，支持选择所属数据板块下的已上线的业务实体。若没有合适的业务实体，可单击**新建业务实体**进行创建，如需创建请参见[创建及管理业务实体](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/create-and-manage-a-business-entity)。 若该字段已关联业务实体，更换数据板块后，业务实体需重新选择。 - 当表类型为维度逻辑表、事实逻辑表时展示在**研发** > **开发**该逻辑表编辑页面关联维度所配置的关联实体，单击关联实体后的**查看**图标，查看每个关联实体信息，包括角色名称（如买家、卖家角色的名称）、实体名称（编码）、描述、所属板块、主题域，同时，您可以执行如下操作： - **查看详情**：单击跳转至**规划** > **数据架构**该业务实体页签下查看详细信息。 - **查看实体关系图**：单击打开**实体关系图**面板，查看以该业务实体为中心的实体关系图。 |                                                              |
| **数据分类**         | 需要开通数据安全模块，支持查看字段的数据分类信息。           |                                                              |
| **数据分级**         | 需要开通数据安全模块，支持查看字段的数据分级信息。您可以单击数据分级后的**排序**![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3372971271/p807599.png)图标，字段将根据数据分级进行排序。 |                                                              |

## **智能生成**

Dataphin基于AI大模型能力智能生成表和字段的基本信息，并可从表资产中准确探索有业务价值的指标以及为指标资产生成全面、准确的信息，极大地提升数据配置效率。

**说明**

-   智能生成页面的第一行为已存在的配置信息，第二行为智能生成的配置信息，仅可对已生成的智能配置信息进行修改。
    
-   智能生成字段/技术指标数量过多，需等待较长的时间，请耐心等待。
    

1.  在**准资产**页签，单击目标资产操作列下的**编辑**图标，进入资产编辑页面。
    
2.  在资产编辑页面，单击右上角的**智能生成**按钮，跳转至智能生成页面，系统将为您生成当前资产的基本信息（技术指标资产仅支持编辑基本信息）、字段信息和表相关指标信息，您可以修改已生成的信息。

    -   **终止**：停止正在生成中的相关指标/全部信息。
        
    -   **继续生成**：将继续智能生成停止的相关指标/全部信息。
