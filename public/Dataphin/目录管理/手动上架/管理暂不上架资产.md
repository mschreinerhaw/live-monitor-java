# 管理暂不上架资产

您可以将无需上架的资产指定为暂不上架资产，如需上架资产目录并进行消费，支持将暂不上架资产变更为准资产后，再上架资产目录，并可设置维护权限等操作。

## **权限说明**

-   超级管理员、运营管理员和具有**上架管理**\-**管理**权限的自定义全局角色支持新建及管理暂不上架资产。
    
-   业务系统的运营人员、具有资产维护权限的维护人员支持管理自己所负责的业务系统下的资产。
    

## **操作步骤**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  在左侧导航栏选择**手动上架** > 对应资产类型，再单击**暂不上架资产**，进入**暂不上架资产**页签。
    
    暂不上架资产列表为您展示暂不上架的资产，包括资产的名称、关联技术指标、资产来源、归属目录、信息是否完善、暂不上架时间、暂不上架原因及操作人信息。
    
    -   **资产来源**：为您展示资产对象所属的来源；其中：Dataphin数据板块下仅包含逻辑表；API资产的资产来源为数据服务项目；Quick BI仪表板的资产来源为Quick BI工作空间。
        
    -   **信息是否完善**：为您展示资产的必填属性是否填写完整及是否合法。
    
3.  （可选）您可以根据指标类型、资产来源、归属目录、信息是否完善、暂不上架时间、暂不上架原因进行精确筛选，也可以根据指定条件搜索资产。
    
4.  在目标资产操作列下可进行如下操作。不同类型资产可执行的操作相同，下面以**表资产**为例进行说明。单次批量操作最多可选择10000条资产。
    
    | **操作项**       | **描述**                                                     |
    | ---------------- | ------------------------------------------------------------ |
    | **查看**         | 单击操作列下的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9404276271/p837716.png)图标，查看资产的信息，您可以在右上角选择**隐藏只读属性**，聚焦于支持编辑的信息。 |
    | **变更为准资产** | 单击**变更为准资产**，支持将资产修改为准资产状态。           |
    | **设置维护权限** | 业务指标不支持该操作。 - **单个操作**：单击**设置维护权限**，支持配置资产的维护人员，负责资产的日常维护，包括修改信息及上下架。 - **批量操作**：单击底部**设置维护权限**，支持批量追加或修改维护人员。 - **修改**：将选中资产的维护人员替换为本次指定的维护人员。 - **追加**：保留选中资产的已配置的维护人员并追加本次指定的维护人员。 |
    | **查看变更记录** | 为您展示单个资产最近30天的变更记录，包括属性变更、状态变更等，以便追溯。 |
    | **导出资产**     | 单击底部的**导出资产**，支持批量导出列表选中的资产。对于当前操作人没有维护权限的资产，系统会自动跳过。 如果导出的资产内容较多，文件生成需要等待一段时间，您可以在**批量管理**\\-**批量操作记录**中查看本次导出操作信息，并下载导出文件。 |
    
    若资产被删除，仅支持查看。
