# 批量导入导出资产

支持批量下载当前资产配置的Excel模板，更新后再重新批量导入Dataphin系统，实现资产详情的快速更新，从而提升协同操作效率和资产盘点效率。本文为您介绍如何批量配置资产。

## **前提条件**

-   建议先完成资产目录规划和资产对象属性定义后，再导出需要更新资产的最新配置文件。目录规划和属性定义请参见[创建及管理资产专题](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/create-an-asset-topic-1)、[创建及管理资产对象、API、仪表板属性](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/create-and-manage-r-d-object-properties)。
    
-   表和仪表板资产导出的文件为一个压缩文件，表资产文件包括表、字段和技术指标资产；仪表板资产文件包括仪表板和图表资产。
    

## **使用限制**

-   仅智能研发版、资产运营-高级版/标准版支持批量导入或导出资产。
    
-   文件导入或导出任务的最大并发数为2。
    
-   不同资产类型保留最近导入或导出的50条记录。
    
-   仅导出成功状态的记录支持下载导出文件。
    
-   字段的展示名称/描述支持引用DDL注释和自定义，需要将当前表的全部字段统一配置为引用DDL注释或自定义。
    

## **权限说明**

-   超级管理员、运营管理员和具有**上架管理**\-**管理**权限的自定义全局角色支持查看导入详情、下载导出记录。
    
-   普通成员支持导入导出自己有维护权限的资产。
    

## 批量导入资产

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
2.  在左侧导航栏选择**手动上架** > 选择对应资产类型，进入**准资产**页签。
    
3.  在**准资产**页签，单击右上角的**批量管理**\-**批量导入资产**。业务指标、技术指标、API、仪表板资产与表资产的导入方式一致，下面以**表资产**为例。
    
4.  在**批量导入资产**对话框中，配置参数。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **导入内容** | 当资产类型为表或仪表板时支持此配置项。 - **表**：支持选择导入**表**或**字段**。建议先导入表，再导入字段。 - **仪表板**：支持选择导入**仪表板**或**图表**。建议先导入仪表板，再导入图表。 |
    | **上传文件** | - 仅支持上传.xlsx类型的文件，如果文件中包含多个工作表，系统仅解析第一个工作表的内容。 - 文件大小不超过100M。 - 填写模板时，请参照模板说明进行填写，导入过程中将进行唯一性、规范性校验。 - **工作表详情**：请勿删除说明行和标题行，系统将从第3行开始解析。请勿修改列名或删除列，否则可能会导致模板无法解析，从而导入失败。填写前请删除示例行。 - **列值为空处理**：如果是必填属性则更新失败，保留已有配置不更新；如果是非必填属性则会清空原有配置后保存，如果暂时不需要更新某列信息，请保留导出配置或删除该列。 - 部分属性的更新失败不影响其他校验通过的属性的更新，上架前请检查资产的最新配置并结合导入记录的异常提示进行修改。 - 导入内容为**表**，配置文件还需注意如下填写说明。 根据配置文件的内容按序逐行解析，建议先导入表、再导入字段或技术指标；归属同一张表的字段建议连续放置便于解析，否则配置信息冲突时无法提前检测，可能会被后续记录覆盖更新。 - 导入内容为**字段**，配置文件还需注意如下填写说明。 - 根据配置文件的内容按序逐行解析，建议先导入表、再导入字段；归属同一张表的字段建议连续放置便于解析，否则配置信息冲突时无法提前检测，可能会被后续记录覆盖更新。 - 同一张表的维度字段对应的业务实体所属板块需要相同，统计周期需要提前在**规划** > **公共定义**部分定义。 - 导入内容为**技术指标**，配置文件还需注意如下填写说明。 根据配置文件的内容按序逐行解析，如果需要更新统计粒度等信息，建议先导入表、再导入技术指标。 - 导入内容为**业务指标**，配置文件还需注意如下填写说明。 通过批量导入完成业务指标的批量新建。若某一行业务指标的GUID为空，则该条记录视为新建。 - 导入内容为**仪表板**，配置文件还需注意如下填写说明。 根据配置文件的内容按序逐行解析，建议先导入仪表板、再导入图表；归属同一张仪表板的图表建议连续放置便于解析，否则配置信息冲突时无法提前检测，可能会被后续记录覆盖更新。 |
    
5.  文件上传后，系统将根据上传文件的规范进行校验，校验成功后，单击**开始导入**，对本次导入的文件根据不同资产类型进行不同校验。
    
    -   **表**：
        
        -   **表GUID**：校验本次导入文件的表GUID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括资产的展示名称、展示名称值配置、描述、描述值配置、归属目录ID、资产标签、自定义属性（包括引用系统属性值、超链接）、使用说明是否开启、权限控制是否合规（包括可见范围、数据探查报告、血缘关系、质量概况、元数据变更记录））。
        
    -   **字段**：
        
        -   **字段GUID**：校验本次导入文件的字段GUID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括资产的展示名称、字段展示名称值配置、描述、字段描述值配置、业务类型、关联实体所属板块、关联实体名称）。
        
    -   **技术指标**：
        
        -   **技术指标GUID**：校验本次导入文件的技术指标GUID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括资产的展示名称、描述、归属目录ID、资产标签、自定义属性（包括引用系统属性值、超链接）、技术属性、使用说明是否开启、权限控制是否合规（包括可见范围、数据探查报告、血缘关系、质量概况、指标关系图））。
        
    -   **业务指标**：
        
        -   **业务指标GUID**：校验本次导入文件的业务指标GUID是否存在、是否为空（若为空，则表示新建）。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括资产的名称、展示名称、归属目录ID、资产标签、描述、指标口径、相关业务指标、指标关系图是否开启、指标关系表达式、业务负责人、自定义属性（包括引用系统属性值、超链接）、关联技术指标、使用说明是否开启、权限控制是否合规（包括可见范围））。
        
    -   **API**：
        
        -   **API ID**：校验本次导入文件的API ID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括资产的API ID、归属目录ID、资产标签、描述、使用说明是否开启、权限控制是否合规（包括可见范围））。
        
    -   **仪表板**：
        
        -   **仪表板GUID**：校验本次导入文件的仪表板GUID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括归属目录ID、资产标签、描述、使用说明是否开启、权限控制是否合规（包括可见范围））。
            
        -   **图表所属仪表板GUID**：校验本次导入文件的图表所属的仪表板GUID是否存在、是否为空。
            
        -   **资产信息**：校验本次导入文件的配置信息是否合法（包括图表ID、展示名、描述）。
    
6.  校验通过后会自动进入导入结果页面。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **导入结果** | 支持查看导入进度、导入结果、执行时间。                       |
    | **异常记录** | - 为您展示校验不通过而无法导入的资产记录。 - **表/字段/技术指标**：包括记录在原始文件中对应的行号、类型（仅业务指标展示）、表/字段/技术指标/业务指标的名称、表/字段/技术指标/业务指标GUID、所属表名称（仅字段和技术指标资产展示）和异常提示。 - **API**：包括记录在原始文件中对应的行号、API的名称、API ID和异常提示。 - **仪表板**：包括记录在原始文件中对应的行号、仪表板/图表所属的仪表板的名称、仪表板GUID（仅仪表板展示）、图表ID（仅图表展示）、图表名称（仅图表展示）和异常提示 - 在**异常记录**页签下，可单击**下载异常记录**，将校验不通过的原始记录下载成文件，并自动添加**异常提示**列，您可根据异常提示修改后重新导入。 |
    
    导入完成后，若您想快速查看导入结果，可关闭导入配置对话框，关闭对话框不会终止导入任务的执行。您可以在**准资产**页签，单击**批量管理**\-**批量操作记录**，并选择需要查看的历史导入记录。
    

## 批量导出资产

**说明**

-   仅支持导出当前账号有维护权限的资产。
    
-   表和仪表板资产导出的文件为一个压缩文件，表资产文件包括表、字段和技术指标资产；仪表板资产文件包括仪表板和图表资产。
    

1.  在**准资产**页签，单击**批量管理**\-**导出全量资产**或在资产列表选中资产后，单击底部的**导出资产**。业务指标、技术指标、API、仪表板资产与表资产的导出方式一致，下面以**表资产**为例。
    
2.  在**批量导出资产**对话框中，配置参数。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **资产来源** | 您可选择基于元数据采集任务定时采集或从项目计算源自动采集的资产。特殊的是，Dataphin数据板块下仅包含逻辑表和建模指标，支持多选。 业务指标无需配置此项。 |
    | **资产状态** | 选择需导出的资产范围，支持选择准资产、已上架资产、暂不上架资产，并支持选择更细粒度的资产进行导出。 |
    
3.  单击**确定**，完成资产的导出。
    
    如果导出的资产内容较多，文件生成需要等待一段时间，您可以在**批量管理**\-**批量操作记录**中查看本次导出操作信息，并下载导出文件。
    

## 资产批量操作记录

支持查看每个资产的导入、导出操作记录详情。

1.  在**准资产**页签，单击**批量管理**\-**批量操作记录**。业务指标、技术指标、API、仪表板资产与表资产的批量操作查看方式一致，下面以**表资产**为例。
    
2.  在**批量操作记录**面板，查看资产的导入、导出操作记录。
    
    -   **批量导入**：为您展示资产导入的操作记录详情，包括序号、创建人、资产类型（仅表/仪表板资产展示该信息）、导入的状态、创建时间、结束时间信息。同时，您可单击操作列下的**查看**图标，查看导入详情。
        
    -   **批量导出**：为您展示资产导出的操作记录详情，包括序号、创建人、导出范围（表/技术指标/API/仪表板资产展示该信息）、导出的状态、创建时间、结束时间信息。同时，您可单击操作列下的**下载**图标，下载导出文件至本地。
