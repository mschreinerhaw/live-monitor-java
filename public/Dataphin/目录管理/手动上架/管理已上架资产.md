# 管理已上架资产

已上架资产列表为您展示上架的准资产，您可以在资产目录中查询，并可以对已上架资产执行编辑、下架、设置维护权限等操作。本文为您介绍如何查看及管理已上架资产。

## **权限说明**

-   超级管理员、运营管理员和具有上架管理权限的自定义全局角色支持新建、下架及管理已上架资产。
    
-   业务系统的运营人员、具有资产维护权限的维护人员支持管理自己所负责的业务系统下的资产。
    

## **操作说明**

1.  在Dataphin首页，选择顶部菜单栏的**资产** > **目录管理**。
    
2.  在左侧导航栏选择**手动上架** > 对应资产类型，再单击**已上架资产**。
    
    已上架资产列表为您展示已上架的资产，包括：资产名称、指标类型、关联技术指标、资产来源、归属目录、最新状态、信息是否完善、需重新上架、变更类型、资产上架时间、最新数据时间、最近更新时间等信息。
    
    -   **已删除资产**：若该资产在元数据清单中已删除，通过黄色图标进行标识，您可以单击**下架**，从已上架资产列表中移除。
        
    -   **资产来源**：为您展示资产对象所属的来源；其中：Dataphin数据板块下仅包含逻辑表；API资产的资产来源为数据服务项目；Quick BI仪表板的资产来源为Quick BI工作空间。
        
    -   **最新状态**：针对最新状态为上架失败的资产，您可以将鼠标悬停状态后的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0746511471/p913068.png)图标，查看上架失败的原因，修改后重新上架资产。
        
    -   **信息是否完善**：为您展示资产的必填属性是否填写完整及是否合法。
        
    -   **资产最新状态**：不同状态可执行的管理操作有所不同，详见下方表格。
        
        | **资产状态**                  | **操作项**                                                   |
        | ----------------------------- | ------------------------------------------------------------ |
        | **已上架**                    | 查看、编辑、下架、设置维护权限、查看变更记录。               |
        | **更新待上架**                | 查看、编辑、上架、下架、设置维护权限、查看变更记录。         |
        | **下架失败**/**更新上架失败** | 查看、编辑、上架（对比线上已更新支持该操作）、下架、设置维护权限、查看变更记录。 |
        | **更新上架中/下架中**         | 处于变更中状态的资产不可操作，避免产生预期外的变更。         |
    
3.  （可选）您可以根据指标类型、资产来源、最新状态、归属目录、信息是否完善、资产上架时间、最新数据时间、最近更新时间、需重新上架、变更类型进行精确筛选，也可以根据指定条件搜索资产。
    
4.  在目标资产操作列下可进行如下操作。不同类型资产可执行的操作相同，下面以**表资产**为例进行说明。单次批量操作最多可选择10000条资产。
    
    | **操作项**       | **描述**                                                     |
    | ---------------- | ------------------------------------------------------------ |
    | **查看**         | 跳转至**资产目录** > **资产详情**页面，查看资产的更多详细信息。 |
    | **编辑**         | 支持修改资产的基本信息、字段信息（仅表资产支持）并配置资产的可见权限，详情请参见[编辑资产信息](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#6ca7f69f10265)。 |
    | **上架**         | 单击**上架**或选中资产后，单击底部的**上架**，可将资产上架到资产目录前台展示。若当前资产配置了审批规则，则上架需要经过审批，审批备注不超过100个字符，审批通过后可上架到资产目录前台；若当前资产未配置审批规则，则可直接上架。 **说明** - 仅必填信息配置完整的资产支持上架。 - 当表资产上架后，该表下的自定义指标资产才能上架。 - 仅具有资产维护权限的用户及角色可上架资产。 - 已上架状态的业务指标名称需在租户内唯一。 |
    | **下架**         | 单击**下架**或选中资产后，单击底部的**下架**，下架单个资产或批量下架资产。若当前资产配置了审批规则，则下架需要经过审批，审批备注不超过100个字符，单击**确定**。审批通过后，下架成功的资产会移至准资产列表，且无法在资产目录中继续查看；下架失败的资产会变为**下架失败**状态，但仍可在资产目录中查看；若当前资产未配置审批规则，则可直接下架。 **说明** - 单次批量操作最多可选择10000条资产。 - 表资产下架将同步下架该表资产中的指标资产，无表资产维护权限的资产将被跳过。 |
    | **设置维护权限** | 业务指标不支持该操作。 - 单击**设置维护权限**，支持配置资产的维护人员，负责资产的日常维护，包括修改信息及上下架。 - 最多选择10000条资产。单击底部**设置维护权限**，支持批量追加或修改维护人员。 - **修改**：将选中资产的维护人员替换为本次指定的维护人员。 - **追加**：保留选中资产的已配置的维护人员并追加本次指定的维护人员。 |
    | **查看变更记录** | 为您展示单个资产最近30天的变更记录，包括属性变更、状态变更等，以便追溯。 |
    | **修改属性**     | 选中资产后，单击底部的**修改属性**，在批量修改资产属性页面，修改资产的属性。配置详情与准资产的批量修改属性一致，详情请参见[修改属性](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#8f59e9e199gx6)。 与编辑单个资产相比，部分属性支持更灵活的配置。 |
    | **智能生成**     | 仅表/技术指标资产支持该操作。仅针对可编辑的资产进行智能生成，选中需智能生成的资产，单击底部的**智能生成**，在新建智能生成任务对话框中，配置信息，详情请参见[智能生成](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/intelligent-generation)。 |
    | **导出资产**     | 单击底部的**导出资产**，支持批量导出列表选中的资产。针对当前操作人没有维护权限的资产，系统会自动跳过。 如果导出的资产内容较多，文件生成需要等待一段时间，您可以在**批量管理**\\-**批量操作记录**中查看本次导出操作信息，并下载导出文件。 |
    
    若资产被删除，仅支持查看。
