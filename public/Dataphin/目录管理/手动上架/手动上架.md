# 手动上架

Dataphin支持手动上架并管理资产，对资产识别后决定是否上架，以便后续在资产目录中消费该资产。

## 前提条件

-   **表资产**：已开通智能研发版或已购买资产运营-标准版增值模块。
    
-   **指标资产**/**仪表资产**：已开通智能研发版或已购买资产运营-高级版增值模块。
    
-   **API资产**：已开通智能研发版或已购买资产运营-高级版增值模块，并购买数据服务功能。
    

## 功能概述

仅**已上架**状态的资产可以在资产目录中搜索、查看。从数据源采集到的资产和从数据中台获取的资产最初会显示在准资产列表中。上架资产前，您需要完善属性，包括设置专题和目录以及指定可见范围等。此外，针对无需上架的资产，您可变更为**暂不上架**状态以便区分。

## **核心概念**

| **名词**         | **说明**                                                     | **相关文档**                                                 |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **准资产**       | 通过采集适配器指定数据源采集的资产和从数据中台获取的资产，您可以根据业务场景判断是否上架，如需上架，需完善资产属性，包括设置上架的目标专题及目录、设置资产的可见范围以及是否允许查看数据探查报告；如果不适合上架，则可将资产设置为暂不上架。 | [管理准资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets) |
| **已上架资产**   | 已经上架到资产目录前台，可供查看或申请消费的资产。支持对已上架资产执行编辑、上架、下架、设置维护权限等操作。 | [管理已上架资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-on-shelf-assets) |
| **暂不上架资产** | 您可以将无需上架的资产指定为暂不上架资产，如需上架资产目录并进行消费，支持将暂不上架资产变更为准资产后，再上架资产目录，并可设置维护权限等操作。 | [管理暂不上架资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-assets-that-are-not-on-the-shelves) |
