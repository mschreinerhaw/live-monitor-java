# 批量导入导出资产专题

可基于系统模板，使用Excel填写目录配置信息后批量上传，从而将线下已梳理的内容导入Dataphin，实现资产专题和目录的快捷创建，降低人工操作成本；也可以将Dataphin中的资产专题批量导出并下载至本地。本文为您介绍如何批量配置资产专题。

## **使用限制**

-   仅智能研发版、资产运营-高级版/标准版支持批量导入或导出资产专题。
    
-   文件导入或导出任务的最大并发数为5。
    
-   仅导出成功状态的记录支持下载导出文件；系统将保存导出文件7天，超过7天将无法下载。
    
-   仅保留最近50条资产专题的导入或导出记录。
    

## **权限说明**

-   超级管理员、运营管理员和具有**目录规划**\-**管理**权限的自定义全局角色支持批量导入或导出资产专题、查看导入详情或下载导出文件。
    
-   专题管理员支持在自己所管理资产专题下批量导入或导出资产专题、查看导入详情或下载导出文件。
    

## **批量导入资产专题**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
    您可以为不同的资产批量导入各自的资产专题及目录，指标专题、API专题、仪表板专题与表专题的导入方式一致，下面以**表专题**为例。
    
2.  在左侧导航栏选择**目录规划** > **目录规划**，在目录规划页面，单击**表目录**页签。
    
3.  在**表目录**页签，单击专题列表底部的**批量管理**\-**批量导入目录**或在目录列表单击**编辑目录**按钮后的下拉箭头选择**批量导入目录**。
    
4.  在**批量导入目录**对话框中，配置参数。
    
    | **参数**                        | **描述**                                                     |
    | ------------------------------- | ------------------------------------------------------------ |
    | **STEP 1 下载配置模板**\\--新增 |                                                              |
    | **配置模板**                    | 若没有模板，可单击文件名称或下载![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0375247371/p886770.png)图标下载.xlsx文件；若已有模板，可直接执行上传配置文件并导入。 |
    | **STEP 1 下载配置模板**\\--更新 |                                                              |
    | **资产专题**                    | 支持选择当前账号有管理权限的资产专题，支持多选。             |
    | **目录版本**                    | 根据所选资产专题导出目录模板，支持选择编辑中（自动过滤已删除的目录）、已发布2种目录版本。 - **编辑中（自动过滤已删除的目录）**：导出所选资产专题下非删除态（即不包含待发布（已删除）、已驳回（已删除））的目录。 - **已发布**：导出所选资产专题下已发布的目录。 |
    | **配置模板**                    | 根据所选的资产专题范围生成，单击**生成模板**，生成.xlsx格式的模板文件，可单击文件名称或下载![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0375247371/p886770.png)图标下载.xlsx文件；若已有模板，可直接执行上传配置文件并导入。 |
    | **STEP 2 上传配置文件**         |                                                              |
    | **上传文件**                    | - 仅支持上传.xlsx类型的文件，如果工作表包含多个sheet，系统仅解析第一个sheet的内容。 - 文件大小不超过20M。 - 填写模板时，请参照模板说明进行填写，导入过程中将基于目录名称进行唯一性、规范性校验。 - **工作表详情**：请勿删除说明行和标题行，系统将从第3行开始解析。请勿修改列名或删除列，否则可能会导致模板无法解析，从而导入失败。填写前请删除示例行。 - 如果专题不存在且操作用户有新建专题权限，将自动创建专题，公开范围默认为全员可见，您可在导入后手动调整。暂不支持编辑、删除专题操作。 - 根据配置文件的内容按序逐行解析，如果需要变更多级目录，每一层级的目录都需要填写单独一行记录；如果是新建目录，请先填写父目录、再填写子目录，否则可能会因为找不到父目录而创建失败。如果是删除操作，请先填写子目录、再填写父目录。 - **原目录ID**：系统将根据原目录ID是否为空决定是新增或更新操作，如果原目录ID存在，但是新目录的所有层级目录配置均为空，视为删除目录；如果该目录下已包含资产，会自动解除关联关系。 |
    
5.  文件上传后，系统将根据上传文件的规范进行校验，校验成功后，单击**开始导入**，对本次导入的专题名称、原目录ID、新目录进行解析。
    
    -   **专题名称**：校验本次导入文件的专题名称是否为空。
        
    -   **原目录ID**：校验本次导入文件的原目录ID是否为空。
        
    -   **新目录**：校验本次导入文件的新目录的全路径是否为空、是否完整。
    
6.  校验通过后打开导入结果页面。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **导入结果** | 支持查看导入进度、导入结果、执行时间。                       |
    | **异常记录** | - 为您展示校验不通过而无法导入的目录记录，包括记录在原始文件中对应的行号、专题名称、原始目录名称/ID、新目录名称和异常提示。 - 在**异常记录**页签下，可单击**下载异常记录**，将对应的原始记录下载成文件，根据异常提示修改后重新导入。 |
    
    导入完成后，若您想快速查看导入结果，可关闭导入配置对话框，关闭对话框不会终止导入任务的执行。您可以在**表目录**页签，单击**批量管理**\-**批量操作记录**，并选择需要查看的历史导入记录。
    

## 批量导出资产专题

您可以为不同的资产批量导出各自的资产专题及目录，指标专题、API专题、仪表板专题与表专题的导出方式一致，下面以**表专题**为例。

1.  在**表目录**页签，单击专题列表底部的**批量管理**\-**批量导出目录**或在目录列表单击**编辑目录**按钮后的下拉箭头选择**批量导出目录**。
    
2.  在**批量导出目录**对话框中，配置参数。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **资产专题** | 支持选择当前账号有管理权限的资产专题。若从某个资产专题进行批量导入操作，系统默认填充当前定位的资产专题，支持多选。 |
    | **目录版本** | 根据所选资产专题导出目录，支持选择编辑中（自动过滤已删除的目录）、已发布2种目录版本。 - **编辑中（自动过滤已删除的目录）**：导出所选资产专题下非删除态（即不包含待发布（已删除）、已驳回（已删除））的目录。 - **已发布**：导出所选资产专题下已发布的目录。 |
    
3.  单击**确定**，完成目录的导出。
    
    如果导出的资产专题内容较多，文件生成需要等待一段时间，您可以在**批量管理**\-**批量操作记录**中查看本次导出操作信息，并下载导出文件。
    

## 资产专题批量操作记录

支持查看每个资产专题的导入、导出操作记录详情。指标专题、API专题、仪表板专题与表专题的批量操作查看方式一致，下面以**表专题**为例。

1.  在**表目录**页签，单击专题列表底部的**批量管理**\-**批量操作记录**或在目录列表单击**编辑目录**按钮后的下拉箭头选择**批量操作记录**。
    
2.  在**批量操作记录**面板，查看资产专题的导入、导出操作记录。
    
    -   **批量导入**：为您展示资产专题导入的操作记录详情，包括序号、创建人、导入的状态、创建时间、结束时间信息。同时，您可单击操作列下的**查看**图标，查看导入详情。
        
    -   **批量导出**：为您展示资产专题导出的操作记录详情，包括序号、创建人、导出范围、导出的状态、创建时间、结束时间信息。同时，您可单击操作列下的**下载**图标，下载导出文件至本地。
