# 创建及管理资产专题

资产专题可以统一管理已创建的专题及归属该专题的目录，提高专题管理效率。本文为您介绍如何创建及管理资产专题。

## **权限说明**

-   超级管理员、运营管理员和具有**目录规划**\-**管理**权限的自定义全局角色支持查看、新建及管理资产专题、管理资产专题的目录。
    
-   专题管理员支持管理自己所创建的资产专题，包括编辑专题的基本信息、管理公开范围、管理专题目录等操作。
    

## **新建专题**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **目录管理**。
    
    您可以为不同的资产创建各自的资产专题及目录，指标专题、API专题、仪表板专题与表专题创建方式一致，下面以**表专题**为例进行描述。
    
2.  在左侧导航栏选择**目录规划** > **目录规划**，在**表目录**页签的专题列表中，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8372971271/p806463.png)图标，进入**新建资产专题**对话框。
    
3.  在**新建资产专题**对话框中，配置参数。
    
    | **参数**       | **描述**                                                     |
    | -------------- | ------------------------------------------------------------ |
    | **专题名称**   | 请填写资产专题的名称，专题名称唯一且不超过256个字符。        |
    | **描述**       | 请填写资产专题的背景信息，便于相关人员快速了解。不超过512个字符。 |
    | **专题管理员** | 默认为当前用户，支持选择当前租户下正常状态的用户。           |
    | **公开范围**   | 影响在资产目录是否可见资产专题。支持两种公开范围： - **全员公开可见**：该租户下的所有用户可查看该资产专题。 - **专题管理员及指定人可见**：支持专题管理员以及指定成员查看该资产专题。可通过用户或用户组指定成员。 - **按用户选择**：选择当前租户下正常状态的用户。 - **按用户组选择**：可选择正常且启用状态下的用户组，成员跟随用户组成员的变动自动更新。如需创建用户组，请参见[创建和管理用户组](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/new-and-manage-user-groups)。 |
    | **发布审批**   | 默认**关闭**，开启后，专题需要审批通过后才能发布。           |
    
4.  单击**确定**，完成资产专题的创建。
    

## **管理资产专题**

指标目录、API目录、仪表板目录的页面展示与表目录一致，下面以**表目录**为例进行描述。

您可以在**表目录**页签查看已配置的资产专题并执行编辑、删除等操作。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7446511471/p912784.png)

| **区域** | **描述** |
| ①**专题目录列表** | - 为您展示已配置的资产专题，包括专题的名称、发布状态及描述信息。 选中某一个专题后，右侧列表自动展示归属当前专题的目录。 - **批量管理**：可基于系统模板，填写配置信息后，将线下已梳理的内容批量导入Dataphin，也可将已有的资产目录配置下载至本地，修改后再进行导入，详情请参见[批量导入导出资产专题](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/batch-import-and-export-assets-topic)。 |
| ②**专题基本信息** | 为您展示资产专题的名称、管理员、公开范围、发布审批、描述信息，同时您可以对该资产专题执行编辑和删除操作。 - **编辑**：支持修改专题的所有信息。 - **删除**：资产专题删除后，将同步自动删除资产专题下的目录以及资产所归属的目录，后续您可以通过修改属性重新添加目录。 |
| ③**目录列表** | 为您展示已发布上线目录的发布状态、名称、描述、最近更新人和最近更新时间信息，您可以展开目录查看归属当前目录的子目录信息。同时，您可以执行发布专题或编辑目录操作。 - **发布专题**：若专题处于发布中、待发布状态，单击**发布专题**按钮，系统将统一发布目录编辑态中的最新内容，您可以单击**查看**，进入目录编辑页查看最新目录内容；若专题开启了发布审批，需要审批通过后才能成功发布。 - **编辑目录**：支持在专题下创建目录，详情请参见[管理资产专题的目录](#963972fee250b)。 |

## 管理资产专题的目录

目录能够帮助您对资产进行更细粒度的分析管理，以便更快、更精准地搜索资产并更好地进行资产运营管理。指标目录、API目录、仪表板目录的管理方式与表目录一致，下面以**表目录**为例进行描述。

1.  在Dataphin首页，在顶部菜单栏选择**资产** > **目录管理**。
    
2.  单击左侧导航栏的**目录规划**，再单击**编辑目录**。
    
3.  在**编辑目录**页面，单击**新建一级目录**按钮，创建目录。
    
    | **参数**     | **描述**                                                     |
    | ------------ | ------------------------------------------------------------ |
    | **目录名称** | 填写目录的名称，同一目录下的同级目录不可重名。不超过128个字符。 |
    | **目录描述** | 填写目录的相关描述。不超过128个字符。                        |
    
4.  单击**保存**![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8372971271/p806507.png)图标，完成目录的创建。
    

您可以在编辑目录页面对已创建的目录执行编辑、删除、移动等操作。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7446511471/p912739.png)

| **区域**          | **描述**                                                     |
| ----------------- | ------------------------------------------------------------ |
| ①**搜索区**       | 您可以根据目录的名称或描述搜索目录。                         |
| ②**功能区**       | - **批量编辑**：单击**批量编辑**按钮，将需要修改的内容修改后，单击右上方的**完成编辑**按钮，完成目录的批量修改。当专题目录的申请状态为待发布（已删除）、发布审批中时，不支持批量编辑操作。 - **排序**：单击**排序**按钮，拖拽目录（包括当前目录及其子目录）进行排序或选择**按目录名称正序**、**按目录名称倒序**进行排序，即刻生效，单击右上方的**退出排序**按钮，退出目录的排序页面。 **说明** - 若当前一级目录的状态为发布中或审批中，不支持移动该一级目录及其子目录或将其他目录移至该一级目录及其子目录中；但支持移动该目录下一级子目录间的顺序。 - 若将一级目录及其子目录移动至新的父目录中，该父目录不可为发布中或审批中状态。 - 一级目录顺序发生变更后，需要发布专题才会生效。 - **发布专题**：单击**发布专题**按钮，将未发布的一级目录进行统一发布。如果专题开启了发布审批，需要审批通过后才能成功发布。 - **新建一级目录**：创建一级目录，详情请参见[管理资产专题的目录](#963972fee250b)。 - **刷新**：单击即可刷新目录列表。 - **全部展开**/**全部收起**：单击即可展开或收起目录树。 |
| ③**目录列表**     | 为您展示目录的名称、ID、描述、申请状态以及目录的最近更新人、最近更新时间信息。不同的目录状态支持执行不同的操作，详情请参见[目录操作](#af7c41c61dnzk)。 |
| ④**批量操作区**   | 您可以对目录批量执行移动、发布、删除、恢复删除等操作。       |
| ⑤**退出目录编辑** | 您可以单击**退出编辑**按钮，退出目录的编辑模式；单击**发布并退出**按钮，发布目录后，退出目录的编辑模式。 |

### **目录操作**

**目录状态**

| **目录申请状态**     | **操作项**                                                   |
| -------------------- | ------------------------------------------------------------ |
| **待发布**           | 新建子目录、保存并新建一行、编辑、移动、删除、发布。         |
| **已发布**           | 新建子目录、保存并新建一行、编辑、移动、删除。               |
| **发布审批中**       | 查看审批、撤回审批、恢复删除。                               |
| **已驳回**           | 新建子目录、保存并新建一行、编辑、发布、移动、查看审批、删除。 |
| **待发布（已删除）** | 恢复删除、发布。                                             |
| **已驳回（已删除）** | 查看审批、恢复删除、发布。                                   |

**操作说明**

| **操作项**         | **描述**                                                     |
| ------------------ | ------------------------------------------------------------ |
| **新建子目录**     | 支持创建不超过10级目录。                                     |
| **保存并新建一行** | 保存子目录信息并自动创建同级目录。                           |
| **编辑**           | 支持修改目录的信息。                                         |
| **发布**           | 当目录存在新增、编辑、删除、移动操作变更时，需要将变更目录的所属一级目录发布才能生效。 **说明** - 若存在跨一级目录移动目录时，需要将来源父目录和目标父目录所在的一级目录同时发布。 - 若专题开启审批，审批通过后，目录才能成功发布。 - 批量发布目录，系统将自动忽略发布中、审批中的一级目录以及待发布、待发布（已删除）状态的目录。 |
| **移动**           | 支持移动目录的所属上级目录，暂不支持跨专题移动目录。 **说明** 来源目录和目标目录的状态需为**非**发布中或审批中。 |
| **删除**           | 删除目录后，将同步删除当前目录及其子目录，状态变更为待发布（已删除），单击**发布专题**按钮，完成目录的删除，若专题开启审批，审批通过后，目录才能成功删除。 **说明** - 若目录下存在资产，删除目录后，会造成资产不归属任何目录，您可以修改资产属性后重新上架资产。 - 批量删除目录时，系统会自动忽略发布中或审批中的目录。 |
| **撤回审批**       | 若专题开启了发布审批，支持将提交审批中的目录撤回。           |
| **查看审批**       | 单击**查看审批**图标，跳转至**任务中心**查看该审批单的审批进度。 |
| **恢复删除**       | 恢复删除后，将待发布（已删除）状态的目录及其子目录恢复至删除前的状态。 **说明** 批量恢复删除目录时，系统会自动忽略发布中或审批中的目录。 |

## **后续步骤**

资产专题创建完成后，您可以将资产归类到资产专题的目录，以便于精细化管理资产，详情请参见[管理准资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets)。
