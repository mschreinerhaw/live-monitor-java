# 标准清单

标准清单为您展示已生效的数据标准信息，同时支持搜索、收藏等操作，为您提供快捷、高效的数据查找入口。本文为您介绍如何查看标准详情。

## **前提条件**

已购买数据标准增值服务并且当前租户已开通数据标准模块。

## 标准清单入口

在Dataphin首页，单击顶部菜单栏**资产** > **标准清单**，进入标准清单页面。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6272971271/p807784.png)

## **查看标准清单**

为您展示数据标准信息。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6272971271/p807788.png)

| **区域**          | **描述**                                                     |
| ----------------- | ------------------------------------------------------------ |
| ①**搜索与筛选区** | 您可以根据标准的名称或编码进行模糊搜索，也可以根据负责人进行筛选。 |
| ②**标准列表**     | 为您展示数据标准的名称、编码、状态、版本、生效时间、负责人、标准的最近更新人及最近更新时间、描述信息。 **查看标准详情**：您可以单击标准的名称，查看标准的详细信息，详情请参见[标准详情](#title-t1t-hz6-s58)。 |

## 标准详情![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6272971271/p807804.png)

| **编号**      | **描述**                                                     |
| ------------- | ------------------------------------------------------------ |
| ①**搜索区**   | 支持快捷搜索其他数据标准，选择需要查看的标准，系统将切换至该标准详情页。 |
| ②**概要信息** | 为您展示标准的名称、编码以及描述信息。 **收藏**：单击即可收藏或取消收藏。收藏后，您可以在**个人数据中心**查看收藏的标准，方便后续查询，详情请参见[查看及管理我收藏的资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-my-collection-of-assets)。 |
| ③**资产详情** | 为您展示标准的属性名称、属性英文名、属性值、属性类型、是否必填、取值类型、描述信息。 |
| ④**资产信息** | 为您展示标准的**基础信息**、**使用信息**和**变更信息**。 - **基础信息**：包括标准集、生效时间、负责人、创建人、创建时间信息。 - **使用信息**：包括收藏次数信息，为您展示当前标准的访问页次（PV），每刷新1次增加1次，实时更新。 - **变更信息**：包括最近更新人、最近更新时间信息。 |
