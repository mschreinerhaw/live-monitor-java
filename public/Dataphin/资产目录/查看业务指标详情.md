# 查看业务指标详情

仅已上架的资产可查看资产详情。本文为您介绍如何查看业务指标资产详情。

## 前提条件

已开通智能研发版或已购买资产运营-高级版增值模块。

## 指标资产详情

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **资产目录**。
    
2.  选择**指标**资产页签，单击目标资产的名称，进入资产详情页面。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9437351571/p978527.png)
    
    | **区域**      | **描述**                                                     |
    | ------------- | ------------------------------------------------------------ |
    | ①**概要信息** | 为您展示业务指标的名称、类型、描述、标签信息。同时。您可以对资产执行以下操作： - **搜索资产**：支持快捷搜索其他对象，选择需要查看的资产，跳转至该指标详情页。 - **收藏**：单击即可收藏或取消收藏。收藏后，您也可以在**个人数据中心**查看全量收藏的资产，详情请参见[查看及管理我收藏的资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-my-collection-of-assets)。 - **上架管理**：单击**上架管理**跳转至目录管理下当前指标资产编辑页面，可修改资产的属性信息、使用说明等，详情请参见[编辑资产信息](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#6ca7f69f10265)。 |
    | ②**基本信息** | - **属性信息**：为您展示指标的属性信息，包括指标口径、管理属性。 - **指标口径**：当前指标的开发逻辑，便于业务人员了解指标。 - **业务属性**：为您展示指标的负责人信息。 - **自定义属性**：为您展示在**数仓规划** > **属性管理**所创建的可见于资产目录的属性。其中，自定义属性归属于业务属性、管理属性、技术属性。 - **相关指标**：仅可查看有权限且已上架的相关指标。支持切换查看业务指标和技术指标。 - **业务指标**：为您展示指标的名称、指标口径（仅已上架并有查看权限且已配置指标口径支持查看）、关联关系信息，同时，您可执行如下操作。 - **查看**：跳转至资产目录下该指标的详情页面。 - **查看指标分析图**：需为建模指标且在手动上架的业务指标资产编辑页面开启指标关系图才能查看指标关系。单击查看以当前资产为根节点的指标关系图。 - **技术指标**：为您展示指标的名称、统计粒度、统计周期、指标口径（仅已上架并有查看权限且已配置指标口径支持查看）、描述、所属汇总表、数据类型信息。同时，您可执行如下操作。 **查看**：单击跳转至资产目录下该指标的详情页面。 - **使用说明**：需要在手动上架的业务指标资产编辑页面开启使用说明。为您展示当前资产的使用说明，为数据浏览者和使用者提供信息参考。 - **指标关系**：支持查看以当前节点为起点的完整指标关系图，默认展开10层，可手动展开更多内容，通过指标运算逻辑形成指标关系图，直观呈现各指标间的加工逻辑。如果关系图内部存在重复节点，仅支持展开第一个出现的节点的下游，其他节点不支持重复展开，避免内容错乱或重复。 **查看详情**：单击跳转至**资产** > **资产目录**该指标的详情页面查看更多信息。 |
    | ③**资产信息** | 为您展示业务指标的**用数统计**和**基本信息**等详细信息。 - **用数统计**：包括指标的浏览量和收藏数。 - **浏览量**：为您展示当前指标的访问页次（PV），每刷新1次增加1次，实时更新。仅包含访问资产目录详情页的页次。 - **收藏数**：为您展示当前指标被用户收藏的次数，实时更新。 - **基本信息**：包括初次上架时间、最近上架时间、发布人、归属目录。 |
