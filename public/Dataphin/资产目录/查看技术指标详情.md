# 查看技术指标详情

仅已上架的资产可查看资产详情。本文为您介绍如何查看指标资产详情。

## 指标资产详情

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **资产目录**。
    
2.  选择**指标**资产页签，单击目标资产的名称，进入资产详情页面。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4083048471/p952040.png)
    
    | **区域**      | **描述**                                                     |
    | ------------- | ------------------------------------------------------------ |
    | ①**概要信息** | 为您展示指标的名称、类型、描述、标签信息。同时。您可以对资产执行以下操作： - **查看元数据**：单击跳转至资产清单该指标的详情页。自定义指标不支持该操作。 - **搜索资产**：支持快捷搜索其他对象，选择需要查看的资产，跳转至该指标详情页。 - **收藏**：单击即可收藏或取消收藏。收藏后，您也可以在**个人数据中心**查看全量收藏的资产，详情请参见[查看及管理我收藏的资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-my-collection-of-assets)。 - **申请权限**：单击快捷跳转到当前表的权限申请页面，详情请参见[申请、续期和交还表权限](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/authority-to-apply-renew-and-return-forms)。 - **上架管理**：单击跳转至目录管理下当前指标资产编辑页面，可修改资产的属性信息、使用说明等，详情请参见[编辑资产信息](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#6ca7f69f10265)。 - **反馈质量问题**：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8404276271/p840497.png)图标，选择**反馈质量问题**，您可以通过该功能向当前质量负责人反馈在使用资产过程中遇到的质量问题（如数据产出不稳定、数据内容不准确等），以通知相关负责人及时进行整改，从而推动提升资产可用性和健康度。配置详情请参见[添加并管理问题清单](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-the-governance-workbench)。 您需要开通数据质量模块才能使用反馈质量问题功能。后续，您可以在质量模块的问题清单中查看反馈问题的处理流程和结果。 |
    | ②**基本信息** | - **属性信息**：为您展示指标的属性信息，包括指标口径、业务属性、技术属性。 - **指标口径**：仅建模指标支持查看。为您展示**研发** > **开发** > **指标**中所填写的指标口径。 - **业务属性**：为您展示所属主题域、关联业务指标信息。 **关联业务指标**：单击业务指标的名称跳转至资产目录下该指标的详情页面。 - **技术属性**：为您展示指标类型、资产来源、所属汇总表、所属数据板块、业务限定、原子指标、统计粒度、统计周期、数据类型、度量单位、是否可累加信息。 - **所属汇总表**：为您展示所属表的名称，若您有该表的查看权限，可单击表名称跳转至资产目录下该表的详情页面。 - **原子指标**/**业务限定**：自定义指标不支持该操作。单击指标名称后**查看**图标，查看该指标的计算逻辑。 - **统计粒度**：可查看组成统计粒度的实体名称，单击**查看**图标，查看每个实体的名称、编码、描述、所属板块、主题域信息，同时，您可以执行如下操作： - **查看详情**：单击跳转至**规划** > **数据架构**该业务实体页签下查看详细信息。 - **查看实体关系图**：单击打开**实体关系图**面板，查看以该业务实体为中心的实体关系图。 - **自定义属性**：为您展示在**数仓规划** > **属性管理**所创建的可见于资产目录的属性。其中，自定义属性归属于业务属性、管理属性、技术属性。 - **数据预览**：需申请指标的消费权限后，可以使用数据预览功能，仅查询您有select权限的字段对应的结果，可预览前50条数据。如何申请权限，请参见[申请、续期和交还表权限](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/authority-to-apply-renew-and-return-forms)。 您可以对筛选后的数据根据字段进行查找或筛选、搜索、查看单行数据详情、自适应列宽以及行列转换操作；也可以单击字段后的排序图标，根据字段进行**不排序**、**升序**、**降序**操作，双击字段值，可一键复制字段的值。 - **数据探查**：需开通数据质量功能。若当前指标的所属表有数据探查结果，则可查看指标值字段的探查报告。 - **使用说明**：需要在手动上架的指标资产编辑页面开启使用说明。为您展示当前资产的使用说明，为数据浏览者和使用者提供信息参考。 - **血缘关系**：需要在手动上架的指标资产编辑页面开启血缘关系。为您展示以当前资产为中心节点的直接血缘关系，包括暂未上架和无查看权限的资产对象，但无法查看资产详情。更多详情，请参见[指标血缘](#f82def3f41x6p)。 - **指标关系**：需为建模指标且在手动上架的指标资产编辑页面开启指标关系图。为您展示以当前资产为根节点的指标关系图。更多详情，请参见[指标关系图](#750d184ea4gxz)。 - **质量概况**：需要在手动上架的指标资产编辑页面开启质量概况且已购买数据质量模块。为您展示当前指标的质量分概览、规则校验明细，更多详情，请参见[质量概况](#57c6189674u7f)。 |
    | ③**资产信息** | 为您展示指标的**用数统计**和**基本信息**等详细信息。 - **用数统计**：包括指标的浏览量和收藏数。 - **浏览量**：为您展示当前指标的访问页次（PV），每刷新1次增加1次，实时更新。仅包含访问资产目录详情页的页次。 - **收藏数**：为您展示当前指标被用户收藏的次数，实时更新。 - **基本信息**：包括初次上架时间、最近上架时间、发布人、归属目录。 |
    

### **指标血缘**

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0775247371/p892628.png)

| **区域**        | **描述**                                                     |
| --------------- | ------------------------------------------------------------ |
| **①搜索**       | 支持根据字段的名称关键字搜索当前页面已展开的节点，支持单击搜索框后的左/右箭头，或通过键盘的方向键快捷切换选中的节点。系统默认定位命中的第一个节点。 |
| **②图例**       | 为您展示当前资产的所属层级，包括业务系统（即数据源表、自定义指标所属系统）、数据采集层（即贴源层）、数据加工层（即中间层、逻辑表/建模指标所属系统）、数据应用层（即应用层）、数据服务、报表应用层、通用层。 |
| **③血缘图展示** | 为您展示完整的血缘链路图，可手动展开多级上游或下游。支持根据字段名称关键字进行模糊搜索。特殊的，若存在循环依赖，则无法继续展开，需从起始节点继续查看下游。 - **中心节点**：展示当前指标名称，并在节点左上角用**当前指标**进行标识。 - **指标详情**：鼠标悬停节点查看指标详情，包括指标的类型、名称、资产来源、所属表、统计粒度、归属目录、资产标签、描述。特殊的，中心节点不展示指标详情。 **说明** - 仅已上架、有查看权限并在资产编辑中开启血缘关系的指标支持查看血缘关系。 - 支持单击指标名称跳转至指标详情页面查看血缘关系。 - 支持单击指标的所属表名称跳转至资产详情页面查看血缘关系。 - **节点连线**：若有资产跨层依赖，则用实线表示；若资产为自依赖，则用虚线表示。 ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0775247371/p891744.png) |

### **指标关系图**

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4083048471/p952158.png)

| **区域**        | **描述**                                                     |
| --------------- | ------------------------------------------------------------ |
| ①**图例**       | 为您展示指标关系图的指标类型，包括原子指标、衍生原子指标、派生指标、衍生指标、注册上挂指标。 |
| ②**指标关系图** | 为您展示以当前指标为根节点的指标加工逻辑关系图。鼠标悬停指标名称为您展示指标的名称、度量单位、指标口径信息。 **查看详情**：仅已上架且有查看权限的指标支持操作。不同类型的指标可查看不同信息。 - **原子指标**/**衍生原子指标**：为您展示指标的名称、指标口径、描述、指标类型、度量单位以及该指标的计算逻辑。 - **派生指标**/**衍生指标**/**注册上挂指标**：为您展示指标的名称、指标口径、描述、统计粒度、统计周期、原子指标、业务限定、指标类型、度量单位、计算逻辑信息。 **说明** - 仅已上架并有查看权限且已配置指标口径才能查看指标口径。 - 针对指标关系图上的每个指标（派生指标/衍生指标/注册上挂指标），需在上架时开启计算逻辑，才能在指标关系图上查看计算逻辑。 |

### **质量概况**

![image](data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNkYAAAAAYAAjCB0C8AAAAASUVORK5CYII=)

| **区域**          | **描述**                                                     |
| ----------------- | ------------------------------------------------------------ |
| ①**质量分雷达图** | 为您展示当前指标的质量特性雷达图、整体质量评分及校验通过规则/计分规则。 - **六性雷达图**：鼠标悬停质量特性名称，可查看该特性的质量分。 - **整体质量评分**：统计当前字段最近1次对应已创建且运行完成过的规则。计算口径为： `质量分=sum（每条规则权重 * 该规则质量分）/ sum（每条规则权重）` - **校验通过规则**：当前指标参与质量分计算且最近1次校验状态通过的规则总数。 - **计分规则**：当前指标参与质量分计算的规则总数。 |
| ②**评分列表**     | 为您展示当前指标的质量特性评估明细，包括评估维度、质量分、校验通过规则/计分规则、通过率、表级及字段级规则数。 - **质量分**：基于当前指标参与质量分计算且规则分类包含对应评估维度相关的规则的最近1次运行状态进行统计。指定评估维度的规则的计算口径为： `质量分=sum（每条规则权重 * 该规则质量分）/ sum（每条规则权重）` - **校验通过规则**：当前指标创建的规则分类包含对应维度、参与质量分计算且最近1次校验状态**通过**的规则总数。 - **计分规则**：当前指标创建的规则分类包含对应维度、参与质量分计算的规则总数。 - **通过率**：即`校验通过规则/计分规则`的百分比。 |
| ③**规则校验明细** | 需要在手动上架的表资产编辑页面开启规则校验明细。您可以指定日期查看规则校验明细，或快捷筛选**异常结果**，或根据校验状态进行筛选，或通过规则的名称关键字进行搜索。 - 若为自定义指标，统计数据包含该指标的来源字段创建的监控规则的最近30天执行完成的校验记录，将根据规则计分方式和校验结果进行统计。 - 若为建模指标，统计数据包含将其作为**指标**创建的监控规则和作为所属表的普通字段创建的监控规则的最近30天执行完成的校验记录，将根据规则计分方式和校验结果进行统计。 |
