# 查看数据表资产详情

仅已上架的资产可查看资产详情。本文为您介绍数据中台表和其他系统表的资产详情。

## **表资产详情**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **资产目录**。
    
2.  选择表资产页签，单击目标资产的名称，进入资产详情页面。
    
    Dataphin表资产详情与其他系统表资产详情展示基本一致，下图以**Dataphin数据表**为例。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8793529371/p890864.png)
    
    | **区域**      | **描述**                                                     |
    | ------------- | ------------------------------------------------------------ |
    | ①**概要信息** | 为您展示数据表的名称、类型、最高涉密等级（需要购买数据安全功能）、标签、统计粒度、描述信息。同时，您可以对资产执行以下操作： - **查看元数据**：单击**查看元数据**，跳转至资产清单该表的详情页。 - **搜索资产**：支持快捷搜索其他数据表，选择需要查看的数据表，跳转至该数据表详情页。 - **统计粒度**：当表类型为汇总逻辑表且该表存在业务类型为维度的字段时展示，支持查看每个统计粒度信息，包括统计粒度名称、编码、描述、所属板块、主题域信息，同时，您可以执行如下操作： - **查看详情**：单击跳转至**规划** > **数据架构**该业务实体页签下查看详细信息。 - **查看实体关系图**：单击打开**实体关系图**面板，查看以该业务实体为中心的实体关系图。 - **收藏**：单击即可收藏或取消收藏。收藏后，您也可以在**个人数据中心**查看全量收藏的资产，详情请参见[查看及管理我收藏的资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-my-collection-of-assets)。 - **上架管理**：单击跳转至目录管理下当前表资产编辑页面，可修改资产的属性信息、使用说明等，详情请参见[编辑资产信息](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#6ca7f69f10265)。 - **反馈质量问题**：您可以通过该功能向当前质量负责人反馈在使用资产过程中遇到的质量问题（如数据产出不稳定、数据内容不准确等），以通知相关负责人及时进行整改，从而推动提升资产可用性和健康度。配置详情请参见[添加并管理问题清单](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-the-governance-workbench)。 您需要开通数据质量模块才能使用反馈质量问题功能。后续，您可以在质量模块的问题清单中查看反馈问题的处理流程和结果。 - **查看DDL语句**：当数据表为Dataphin表时，支持查看当前数据表在租户计算引擎中对应的DDL语句。此外，您也可以选择**数据源类型**，单击**生成DDL语句**，系统将生成在指定类型的数据源系统中创建与当前表相同结构的表DDL语句。若您勾选了**自动添加转义符**，生成的DDL语句将根据选择的源类型自动添加对应的转义符，减少系统关键词误转义等问题的发生。部分数据源表，支持查看DDL语句，详情请参见[不同类型的采集源支持的操作](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/operations-supported-by-different-types-of-collection-sources)。 - **导出字段**：导出表中的字段信息至本地，格式类型为CSV，便于其他开发或业务人员快速分析使用。 |
    | ②**基本信息** | - **属性信息**：为您展示数据表的属性信息，包括业务属性、管理属性、技术属性。 - **业务属性**：为您展示所属主题域信息。 - **管理属性**：为您展示归属系统信息。 - **数据源表**：即为采集任务的来源系统。 - **Dataphin表**：展示为-。 - **技术属性**：为您展示资产来源、所属板块/项目空间、数据库类型、所属的项目、表类型、资产引入名称信息。 - **资产来源**： - **数据源表**：来源路径为来源系统 > 数据源名称 > 表所在Schema。 - **Dataphin表**：来源路径为Dataphin > 项目空间类型（逻辑表标识为中间层） > 项目名称/板块名称（板块下仅包含逻辑表）。 - **资产引入名称**： - **数据源表**：数据源表采集任务的名称。 - **Dataphin表**：展示为-。 - **自定义属性**：为您展示在**数仓规划** > **属性管理**所创建的可见于资产目录的属性。其中，自定义属性归属于业务属性、管理属性、技术属性。 - **字段信息**：为您展示字段的序号、名称、数据类型、描述/备注、业务类型/关联实体（维度逻辑表和事实逻辑表仅展示关联实体）、关联标准（需购买数据标准模块）、样例数据（仅开启[数据采样](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/untitled-document-1760410212878#ff1c6fe9aacgu)功能时展示）、质量分（需购买数据质量模块且资产编辑中开启质量概况）、数据分类（需购买数据安全模块）、数据分级（需购买数据安全模块）信息。 - **搜索及筛选**：您可以根据字段的名称、展示名、描述或备注搜索字段，也可根据数据分类、数据分级筛选指定字段（需开通数据安全）。 - **关联实体**：当表类型为维度逻辑表、事实逻辑表时展示在**研发** > **开发**页面下该逻辑表编辑页面关联维度所配置的关联实体，单击关联实体后的**查看**图标，查看每个关联实体信息，包括角色名称（如买家、卖家角色的名称）、实体名称（编码）、描述、所属板块、主题域，同时，您可以执行如下操作： - **查看详情**：单击跳转至**规划** > **数据架构**该业务实体页签下查看详细信息。 - **查看实体关系图**：单击打开**实体关系图**面板，查看以该业务实体为中心的实体关系图。 - **业务类型**：若建模指标已上架，可单击指标名称跳转至**目录管理** > **手动上架** > **技术指标**已上架页面查看详细信息。 - **质量分**：单击质量分值后的**查看**图标，查看当前字段的评估维度和质量分。单击**查看更多**，跳转至当前资产的**质量概况**页签查看质量分概览。 - **数据预览**：若数据表已有采样数据，则优先展示采样数据，也可手动触发查询最新数据；若不存在采样数据，则会自动触发数据预览查询。 您可以对筛选后的数据根据字段进行查找或筛选、搜索、查看单行数据详情、自适应列宽以及行列转换操作；也可以单击字段后的排序图标，根据字段进行**不排序**、**升序**、**降序**操作，双击字段值，可一键复制字段的值。支持查看样例数据和预览数据。 - **样例数据**：当数据采样开关开启且使用配置中数据预览开关开启时支持展示。仅查询您有列级权限且无需脱敏字段对应的样例数据。针对每个字段的采样数据，系统会独立存储和排序展示，但不保证**行**记录的存在性和正确性。 - **数据预览**：若您拥有当前表的查表数据权限，可以使用数据预览功能，仅查询您有select权限（包括字段权限和行级权限）的字段的对应结果，可预览前50条数据。如何申请查询权限，请参见[申请、续期和交还表权限](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/authority-to-apply-renew-and-return-forms)，支持数据预览的数据源请参见[数据探查支持的数据源和计算引擎](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/exploration-partitions-and-scope-supported-by-different-data-sources)。 - **数据探查**：若您开通了数据质量功能，可以为数据源表配置数据探查任务，快速了解数据概况，提前评估数据的可用性和潜在风险，如果您需要开启自动探查，可以在**治理** > **元数据中心** \\> **探查及分析**中开启相应配置，探查任务配置详情请参见[新建数据探查任务](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/new-data-exploration-task)，支持的数据源类型请参见[数据探查支持的数据源和计算引擎](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/exploration-partitions-and-scope-supported-by-different-data-sources)。 - **使用说明**：需要在手动上架的表资产编辑页面开启使用说明。为您展示当前资产的使用说明，为数据浏览者和使用者提供信息参考。 - **血缘关系**：需要在手动上架的表资产编辑页面开启血缘关系。为您展示系统、表、字段的血缘链路图。 - **系统血缘**：包括当前资产的所属板块/项目/来源系统的系统血缘关系。 若来源系统的相关表被删除或相关表的采集任务被删除，则展示为未知来源系统。更多详情，请参见[系统血缘](#4fe293b097uuf)。 - **表/字段血缘**：为您展示以当前资产为中心节点的直接血缘关系，包括暂未上架和无查看权限的资产对象，但无法查看资产详情。更多详情，请参见[表级血缘](#74b8fb75c9kad)、[字段级血缘](#c01c6d3c1c423)。 - **质量概况**：需要在手动上架的表资产编辑页面开启质量概况且已购买数据质量模块。为您展示当前数据表及当前表所属字段的质量分概览、规则校验明细，更多详情，请参见[质量概况](#278bcbf5a5vus)，仅支持数据质量监控的数据表可查看质量概况，支持的数据源请参见[Dataphin支持的数据源](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/dataphin-supported-data-sources)。 - **元数据变更**：需要在**元数据** > **元数据变更记录**页面和在手动上架的表资产编辑页面均开启元数据变更，才能查看元数据的版本变更记录。不同数据源生成元数据版本的说明，请参见[元数据版本](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/appendix-metadata-version-and-attribute-description#a2aded8f1eoue)。 - **版本对比**：支持切换查看历史版本记录，为您展示基本信息、变更字段、全部字段，详情请参见[版本对比](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-metadata-inventory#50377fa026xgr)。 - **下游影响**：仅最近1条变更支持查看下游影响。当变更类型为新增、删除、数据类型、字段键类型、字段是否可为空变更时，可查看下游影响。 |
    | ③**资产信息** | 为您展示数据表的**用数统计**和**基本信息**等详细信息。 - **用数统计**：包括表的浏览量和收藏数。 - **浏览量**：为您展示当前数据表的访问页次（PV），每刷新1次增加1次，实时更新。仅包含访问资产目录详情页的页次。 - **收藏数**：为您展示当前数据表被用户收藏的次数，实时更新。 - **基本信息**：包括初次上架时间、最近上架时间、发布人、归属目录。 |
    

### **系统血缘**

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0775247371/p891412.png)

| **区域**        | **描述**                                                     |
| --------------- | ------------------------------------------------------------ |
| **①搜索**       | 支持根据项目、板块、来源系统的名称关键字搜索当前页面已展开的节点，支持单击搜索框后的左/右箭头，或通过键盘的方向键快捷切换选中的节点。系统默认定位到命中的第一个节点。 |
| **②图例**       | 为您展示当前资产的所属层级，包括业务系统（即数据源表、自定义指标所属系统）、数据采集层（即贴源层）、数据加工层（即中间层、逻辑表/建模指标所属系统）、数据应用层（即应用层）、数据服务、报表应用层、通用层。 |
| **③血缘图展示** | 为您展示当前资产完整的血缘链路图，可手动展开上下游节点，包括当前表及当前指标所属表的直接表级血缘，根据系统汇总进行展示。 - **中心节点**：展示当前资产的所属系统，并在节点左上角用**当前资产所属系统**进行标识。 - **系统详情**：鼠标悬停节点查看系统详情，包括系统的类型（即项目/板块/元数据）、名称、展示名以及系统的管理员（项目）/架构师（板块）/运营人员（系统）、相关资产的数量。 - **系统内表**：鼠标悬停节点并单击右侧下拉箭头，支持通过表名称模糊搜索，单击表名称跳转至当前资产详情页面，查看血缘关系。 **说明** - 仅已上架、有查看权限并在资产编辑中开启血缘关系的表支持查看血缘关系。 - 支持查看归属当前系统下并在当前节点血缘范围内（除当前表之外）表的血缘关系。 - 未知来源系统下被删除的表不支持查看血缘关系。 - **节点连线**：若有资产跨层依赖，则用实线表示；若资产为自依赖（如本系统内一张表依赖另一张表），则用虚线表示。 ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0775247371/p891744.png) |

| ①**筛选**         | 支持根据指定对象（全表或具体字段）筛选需要查看的质量评分概览和规则校验明细。 |
| ②**质量分雷达图** | 为您展示筛选后的指定对象（全表或具体字段）的质量特性雷达图、整体质量评分及校验通过规则/计分规则。 - **六性雷达图**：鼠标悬停质量特性名称，可查看该特性的质量分。 - **整体质量评分**：若对象为表，则统计当前表最近1次所有已创建且运行完成过的规则；若对象为字段，则统计当前字段最近1次对应已创建且运行完成过的规则。计算口径为： `质量分=sum（每条规则权重 * 该规则质量分）/ sum（每条规则权重）` - **校验通过规则**：指定对象（全表或具体字段）参与质量分计算且最近1次校验状态通过的规则总数。 - **计分规则**：指定对象（全表或具体字段）参与质量分计算的规则总数。 |
| ③**评分列表**     | 为您展示筛选后的指定对象（全表或具体字段）的质量特性评估明细，包括评估维度、质量分、校验通过规则/计分规则、通过率、表级及字段级规则数。 - **质量分**：基于指定对象（全表或具体字段）参与质量分计算且规则分类包含对应评估维度相关规则的最近1次运行状态进行统计。指定评估维度的规则计算口径为： `质量分=sum（每条规则权重 * 该规则质量分）/ sum（每条规则权重）` - **校验通过规则**：指定对象（全表或具体字段）创建的规则分类包含对应维度、参与质量分计算且最近1次校验状态**通过**的规则总数。 - **计分规则**：指定对象（全表或具体字段）创建的规则分类包含对应维度、参与质量分计算的规则总数。 - **通过率**：即`校验通过规则/计分规则`的百分比。 |
| ④**规则校验明细** | 需要在手动上架的表资产编辑页面开启规则校验明细。统计数据包含当前数据表创建的监控规则的最近30天执行完成的校验记录，将根据规则计分方式和校验结果进行统计。 您可以指定日期查看规则校验明细，或快捷筛选**异常结果**，或根据校验状态进行筛选，或通过对象、规则的名称关键字进行搜索。 |
