# 查看仪表板资产详情

仅已上架的资产可查看资产详情。本文为您介绍如何查看仪表板资产详情。

## 前提条件

已开通智能研发版或已购买资产运营-高级版增值模块。

## **仪表板资产详情**

1.  在Dataphin首页的顶部菜单栏，选择**资产** > **资产目录**。
    
2.  选择**仪表板**资产页签，单击目标资产的名称，进入资产详情页面。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7456511471/p914278.png)
    
    | **区域**      | **描述**                                                     |
    | ------------- | ------------------------------------------------------------ |
    | ①**概要信息** | 为您展示仪表板的名称、类型、描述、标签信息。同时，您可以对资产执行以下操作： - **搜索资产**：支持快捷搜索其他仪表板，选择需要查看的仪表板，跳转至该仪表板详情页。 - **收藏**：单击即可收藏或取消收藏。收藏后，您也可以在**个人数据中心**查看全量收藏的资产，详情请参见[查看及管理我收藏的资产](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-my-collection-of-assets)。 - **查看仪表板**：单击**查看仪表板**跳转至Quick BI系统该仪表板的详情页面，查看更多信息。 - **编辑**：具有仪表板维护权限的用户支持操作。单击**编辑**跳转至目录管理下当前仪表板资产编辑页面，可修改资产的属性信息、使用说明等，详情请参见[编辑资产信息](https://help.aliyun.com/zh/dataphin/fullmanaged/user-guide/view-and-manage-prospective-assets#6ca7f69f10265)。 |
    | ②**基本信息** | - **属性信息**：为您展示仪表板的属性信息，包括管理属性、技术属性。 - **管理属性**：为您展示资产来源、所属BI工作空间、所属BI目录、负责人、最近更新人、最近更新时间、协同授权策略、是否公开、公开截止时间以及自定义属性信息。 - **资产来源**：通过采集适配器指定Quick BI工作空间采集的已发布的仪表板信息。 - **是否公开**：公开的报表可通过链接直接访问无需登录，而未公开的报表需登录Quick BI系统后进行访问。 - **技术属性**：为您展示来源数据集、图表总数、第三方嵌入状态以及自定义属性信息。 - **图表信息**：为您展示最新采集的图表信息，包括图表的序号、展示名（图表名称）、图表类型、字段类型、字段名称/数据类型以及图表描述信息。 您可以根据图表的名称或描述进行搜索，或根据图表类型、维度、度量进行筛选。 - **使用说明**：需要在手动上架的仪表板资产编辑页面开启使用说明。为您展示当前资产的使用说明，为数据浏览者和使用者提供信息参考。 |
    | ③**资产信息** | 为您展示仪表板的**用数统计**和**基本信息**等详细信息。 - **用数统计**：包括仪表板的浏览量和收藏数。 - **浏览量**：为您展示当前仪表板的访问页次（PV），每刷新1次增加1次，实时更新。仅包含访问资产目录详情页的页次。 - **收藏数**：为您展示当前仪表板在资产目录被用户收藏的次数，实时更新。 - **基本信息**：包括初次上架时间、最近上架时间、发布人、归属目录。 |
